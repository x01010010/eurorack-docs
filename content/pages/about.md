# About This Site

**Modular Field Manual** is an unofficial, community-maintained reference for Eurorack modules. It exists because manufacturer quick-start guides — while accurate — are dense PDFs that aren't fun to navigate at a patch bay at 1 a.m.

Each module gets:

- A plain-language **overview** explaining what the module is and what it's for.
- A complete **panel reference** — every knob, jack, button, and LED.
- Documentation of every **operating mode**, including the menu-driven ones that are easy to miss.
- A handful of **patch recipes** to get you started.
- Full **specifications** in tables you can actually scan.
- **Tips & tricks** distilled from manuals, demos, and use.

## Sources

The content is compiled from official manufacturer documentation, open-source hardware documentation (such as Mutable Instruments by Émilie Gillet), and notes from community demos.

This site is not affiliated with any module manufacturer. Module names and designs are trademarks of their respective holders.

## Built To Grow

The site is content-driven — every page is generated from Markdown and a small JSON manifest per module. Adding a new manual is a 10-minute exercise: drop in a folder, write a few Markdown files, regenerate. See [Adding a New Module](/contributing.html) for the recipe.
