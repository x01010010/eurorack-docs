# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| FREQUENCY | Knob | Filter cutoff frequency |
| RESONANCE | Knob | Filter resonance / Q; self-oscillates at maximum |
| INPUT LEVEL | Knob | Input signal level / drive into the filter |
| SLOPE | Switch | Selects 2-pole (12 dB/oct) or 4-pole (24 dB/oct) for LP output |

## Inputs

| Jack | Function |
| --- | --- |
| IN | Audio input |
| FM1 | Cutoff frequency CV input 1 |
| FM2 | Cutoff frequency CV input 2 |
| V/OCT | 1V/octave pitch tracking input for keyboard/sequencer tracking |
| RES CV | Resonance CV input |

## Outputs

| Jack | Function |
| --- | --- |
| LP | Low-pass output (slope selectable via SLOPE switch) |
| BP | Band-pass output (fixed 2-pole) |
| HP | High-pass output (fixed 2-pole) |
