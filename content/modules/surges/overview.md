# Overview

Surges is a multimode voltage-controlled filter based on the Mutable Instruments Ripples architecture. It provides simultaneous low-pass, band-pass, and high-pass filter outputs with independently switchable slope (2-pole / 4-pole). V/octave pitch tracking allows the filter to track melody lines for consistent timbral character across the keyboard range.

The filter self-oscillates at maximum resonance, producing a clean sine wave at the cutoff frequency. FREQUENCY, RESONANCE, and INPUT LEVEL controls are all present, giving straightforward, direct control over the filter voice.
