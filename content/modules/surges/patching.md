# Patch Recipes

## Classic Subtractive Lead

Patch a sawtooth VCO to IN. Patch an ADSR envelope to FM1. Patch the same V/OCT to both the VCO and the V/OCT input. Set 4-pole LP, moderate RESONANCE. The filter opens and closes with each note, with the cutoff tracking pitch for consistent brightness. Take LP to a VCA.

## Envelope + LFO Layered Filter

Patch a fast envelope to FM1 and a slow LFO to FM2. Each note has a quick attack sweep from the envelope; the LFO adds slow, undulating movement between notes. Set RESONANCE moderate for a "talking" wah-like quality.

## Simultaneous LP + HP Splits

Patch a drum bus or full mix into IN. Take LP out (set low cutoff) to preserve the low end; take HP out (set same or slightly different cutoff) for the high-frequency content. You have a passive crossover from one filter module — send each band to different processing chains.

## BP Wah Effect

Set BP output. Patch a slow envelope or LFO to FM1. The band-pass peak sweeps with the CV, creating a wah pedal character. Increase RESONANCE to tighten the band for a more pronounced wah. Use INPUT LEVEL to drive the filter slightly for added harmonics.

## Self-Oscillating Sine Bass

Turn RESONANCE to maximum (LP mode, 4-pole). Patch V/OCT from a sequencer. The filter sings as a sine oscillator following the pitch sequence. Take LP output through a VCA with a tight envelope for clean plucked sine bass notes.

## Filter FM Percussion

Self-oscillate the filter and patch an audio-rate VCO to FM2. The VCO frequency-modulates the filter's resonant frequency, producing metallic FM percussion timbres. Trigger with a short gate envelope on FM1 for attack. Adjust FM2 level for metallic character.
