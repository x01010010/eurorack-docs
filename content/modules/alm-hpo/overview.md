# Overview

The **HPO** (ALM019) is a no-frills stereo headphone driver from ALM Busy Circuits. Measuring just 2 HP, it squeezes into any leftover gap in a skiff or larger system and provides a clean, low-noise path from modular-level signals to a 3.5 mm stereo headphone jack.

Two audio inputs accept either a true stereo pair or a single mono source normalised across both channels via the left input's normalisation. A single VOLUME knob controls output level, and the circuit is skiff-friendly at only 38 mm depth.

## Key Specifications

| Parameter | Value |
|-----------|-------|
| Width | 2 HP |
| Depth | 38 mm |
| Power | +12 V / 5 mA, −12 V / 5 mA |
| Output | 3.5 mm TRS stereo |
