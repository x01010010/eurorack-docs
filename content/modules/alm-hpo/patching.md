# Patching

## Stereo Output

Patch the left and right outputs of a stereo mixer or effects module into the L and R inputs. Adjust VOLUME to taste and plug headphones into the 3.5 mm jack.

```
Stereo mixer L out → HPO L in
Stereo mixer R out → HPO R in
                     HPO headphone out → headphones
```

## Mono to Stereo

Leave R unplugged. The L input is normalised to both channels, so a mono source is heard equally in both ears.

```
Mono source → HPO L in
              HPO headphone out → headphones (centre-panned mono)
```

## Monitoring Alongside a Line Output

The **HPO** can be patched in parallel with a line-out module by using a mult or stackcable on the audio signal, letting you monitor on headphones while simultaneously recording or sending to speakers.

```
VCO/mixer → mult → HPO L in (monitor)
                 → Line out module (record/speakers)
```
