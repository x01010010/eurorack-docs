# Patch Recipes

## Rotating Filter Sweep

In Single mode with a sine wave, patch outputs 1, 3, 5, and 7 (0°, 90°, 180°, 270°) to the filter cutoff CV of four oscillators or voices. Each filter sweeps at the same rate but at a different phase, creating a slowly rotating, phasing timbre across the mix.

## 8-Destination Modulation Hub

Switch to Multi mode. Patch all eight outputs to eight different CV destinations across your rack — filter, pitch, waveshape, panning, reverb mix, VCA, delay time, resonance. Each destination gets a different waveform shape at the same rate. Adjust WAVE CV with a slow LFO to continuously rotate which shape goes where, generating complex evolving modulation without additional modules.

## Tempo-Synced Chorus Effect

Patch the system clock into SYNC. Route outputs 1 and 5 (0° and 180°) to the pitch CV of two detuned oscillators. The phase opposition causes one oscillator to be going up in pitch while the other goes down, creating a chorus/vibrato effect that's locked to tempo.

## Self-Patching Waveform Evolution (Multi Mode)

In Multi mode, patch output 3 or 4 back into WAVE CV. The LFO continuously changes which waveform appears on which output in a self-modulating loop. The result is an organic, unpredictable modulation source that never quite repeats.

## Phase CV for Spatial Animation

In Single mode, patch a slow secondary LFO into PHASE CV. As the phase window shifts, the "distance" between adjacent outputs contracts and expands — outputs alternately converge on the same phase and spread back apart. Applied to panning or stereo position, this creates a slow breathing spatial effect.

## Freeze as Sample-and-Hold

Set RATE to 12 o'clock (freeze). In Multi mode all outputs hold fixed voltages representing positions on each of the 8 waveforms. Patch a gate source to an external S&H that clocks the RATE knob via a CV-controlled attenuator — or simply turn the rate off and on manually — to "step" through frozen snapshots of the waveforms.
