# Tips & Tricks

- **Never connect the Eurorack power header** — the Expander gets all its power from the Code Source via the ribbon cable. The module's power connector looks similar to a standard Eurorack header; mistaking it for one will permanently damage the Expander. Confirm you are using the ribbon connector before powering up.
- **Remove the security jumper first** — the Code Source ships with a jumper on its ribbon socket to prevent accidental connection. Remove it before attaching the ribbon cable or the connection won't seat correctly.
- **AUTOBEND TIME is the character knob** — short TIME produces a click/snap (great for hi-hat and snare attacks); medium TIME gives a classic chiptune pitch drop; long TIME produces a swooping laser-gun effect. These three regions cover most of the useful percussive palette.
- **Use ENV OUT even when not processing drums** — the autobend envelope is a free, trigger-synced AR envelope. Patch it to a VCA, filter, or reverb mix for rhythm-locked modulation that doesn't require a separate envelope module.
- **CLOCK OUT for tempo-synced triggering** — at very low Code Source RATE, CLOCK OUT becomes a slow gate that can trigger other modules in tempo with the Code Source. Useful for keeping the rest of a patch rhythmically tied to the noise source's own clock.
- **ALT MODE for variety** — if you've exhausted the three main Code Source modes, engage ALT MODE on the Expander. Some X/Y positions that sounded similar in normal mode become very distinct in ALT MODE.
- **Panel placement** — the ribbon cable is short. Mount the Expander directly adjacent to the Code Source in your rack, either left or right. Plan this before drilling rails or building a case.
