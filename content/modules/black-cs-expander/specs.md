# Specifications

## Electrical

| Parameter | Value |
| --- | --- |
| Power source | Via ribbon cable from Black Code Source (not from Eurorack bus) |
| Power (+12 V) | 16 mA |
| Power (−12 V) | 11 mA |
| ENV OUT range | 0–+5 V |
| CLOCK OUT | Reflects Code Source internal clock rate |

## Mechanical

| Parameter | Value |
| --- | --- |
| Width | 6 HP |
| Depth | 33 mm |
| Connection | Ribbon cable (supplied) to Black Code Source rear PCB |

## Compatibility

| Parameter | Value |
| --- | --- |
| Required host | Erica Synths Black Code Source |
| Standalone use | Not supported |
| PSU connection | Not supported — will permanently damage the module |
