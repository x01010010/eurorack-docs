# Overview

Abacus is a dual function generator based on the Make Noise Maths architecture. Each of the two channels provides a voltage-controlled rise/fall generator configurable as an envelope, LFO, slew limiter, or portamento source. A LIN/LOG/EXP switch per channel shapes the curve independently.

Beyond the two function generators, Abacus includes four attenuverters and a sum/OR/invert logic section, making it a compact multi-utility that covers envelope, modulation, signal scaling, and simple logic in one module.
