# Panel & I/O

Abacus has two identical function-generator channels (Ch1 upper, Ch2 lower) plus a shared utility section.

## Channel Controls (per channel)

| Control | Type | Function |
| --- | --- | --- |
| RISE | Knob | Rise (attack) time |
| FALL | Knob | Fall (decay/release) time |
| LIN / LOG / EXP | Switch | Curve shape for both rise and fall segments |
| CYCLE | Button/Switch | Enables free-running LFO mode |

## Utility Controls

| Control | Type | Function |
| --- | --- | --- |
| ATT 1–4 | Knobs (×4) | Four independent attenuverters |

## Inputs

| Jack | Function |
| --- | --- |
| CH1 IN | Gate/trigger input for channel 1 function generator |
| CH2 IN | Gate/trigger input for channel 2 function generator |
| RISE CV (×2) | CV input for RISE time per channel |
| FALL CV (×2) | CV input for FALL time per channel |
| ATT 1–4 IN | Signal inputs for the four attenuverters |

## Outputs

| Jack | Function |
| --- | --- |
| CH1 OUT | Channel 1 function generator output |
| CH2 OUT | Channel 2 function generator output |
| SUM | Sum of CH1 OUT + CH2 OUT |
| OR | Logical OR of CH1 and CH2 gate/trigger signals |
| INV | Inverted output of CH2 |
| ATT 1–4 OUT | Scaled outputs of the four attenuverters |
