# Patch Recipes

## Dual AR Envelope for a Two-Voice Patch

Patch a gate to CH1 IN and CH2 IN. Send CH1 OUT to one voice's VCA and CH2 OUT to another. Set different RISE/FALL times and curve shapes per channel for two independently shaped envelopes from one gate source.

## Paired Attack Envelopes with SUM

Set CH1 to a fast attack (short RISE, medium FALL) and CH2 to a slower attack (medium RISE, longer FALL). Patch SUM to a VCF cutoff — the summed output produces a complex, layered filter sweep that opens quickly and closes slowly.

## LFO with Independent Rate Per Half-Cycle

Enable CYCLE on CH1. Set RISE short and FALL long. The LFO spends little time rising and most of its period falling — useful for a sawtooth-like descending modulation (fast up, slow down or vice versa). Pair with a VCF to create an automatic wah.

## Portamento with Upward-Only Slew

Patch a sequencer pitch CV to CH1 IN (leave CYCLE off). Set FALL to minimum (near-zero). The output glides upward to new notes but snaps downward instantly — the characteristic upward portamento of bass synths.

## Logic Gate from Two Rhythmic Sources

Patch two different gate sequences to CH1 IN and CH2 IN. Take the OR output to a third module — it fires whenever either channel receives a gate. Useful for building irregular rhythmic patterns from two regular sequences.

## Attenuverter as Offset + Scale

Patch a bipolar LFO to ATT1 IN. Turn ATT1 CW to scale it positively (reduce amplitude). Turn CCW to invert it. Use two attenuverter channels with the same source to create inverted modulation pairs for complementary VCF/VCA pushpull.
