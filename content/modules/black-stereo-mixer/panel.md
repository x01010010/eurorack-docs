# Panel & I/O

## Controls

| Control | Function |
|---|---|
| IN1 LEVEL | Ganged L/R volume for stereo input 1. Also selects +6 dB boost when the boost switch is engaged. |
| IN2 LEVEL | Ganged L/R volume for stereo input 2. Also selects +6 dB boost when the boost switch is engaged. |
| IN3 LEVEL | Ganged L/R volume for stereo input 3. |
| IN4 LEVEL | Level for the mono input 4. |
| PAN CV | Sets the panning position of IN4 in the stereo field. Modulated by the PAN CV input jack. |
| IN1 BOOST | Switch — engages +6 dB gain on IN1. |
| IN2 BOOST | Switch — engages +6 dB gain on IN2. |
| IN1/2 MERGE | Switch — sums the L and R inputs of IN1 and IN2 to both channels, allowing a mono signal to appear on both sides. |

## Inputs

| Jack | Description |
|---|---|
| IN1 L / IN1 R | Stereo input 1 — left and right channels. |
| IN2 L / IN2 R | Stereo input 2 — left and right channels. |
| IN3 L / IN3 R | Stereo input 3 — left and right channels. |
| IN4 | Mono input 4, panned by PAN/PAN CV. |
| PAN CV | CV control for IN4 stereo panning. Accepts up to 20 Vptp. |

## Outputs

| Jack | Description |
|---|---|
| OUT L | Stereo bus left output. |
| OUT R | Stereo bus right output. |

## Indicators

- **Signal LEDs** — one for L bus, one for R bus; light up when signal is present in the mix output.

## Panel Notes

- Maximum input signal level is 20 Vptp; inputs are protected against overvoltage.
- The PFL (Pre-fade Listen) expander connects to a header on the rear of the module to add individual channel monitoring without affecting the main mix.
