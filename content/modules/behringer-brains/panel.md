# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| FREQ | Large knob | Coarse pitch (8-octave range) |
| HARMONICS | Knob | Harmonic content / spectral spread (engine-dependent) |
| TIMBRE | Knob | Primary timbre parameter |
| MORPH | Knob | Secondary timbre / waveshape parameter |
| HARMONICS attenuverter | Small knob | Scales HARMONICS CV input |
| TIMBRE attenuverter | Small knob | Scales TIMBRE CV input |
| MORPH attenuverter | Small knob | Scales MORPH CV input |
| BANK | Button | Steps through engine banks; long-press to enter calibration |
| MODEL | Button | Steps through models within current bank |

## Inputs

| Jack | Function |
| --- | --- |
| V/OCT | 1V/octave pitch CV |
| FM | Linear FM input (use with attenuverter) |
| TIMBRE | CV over TIMBRE parameter |
| HARMONICS | CV over HARMONICS parameter |
| MORPH | CV over MORPH parameter |
| MODEL | CV selects engine model (quantised to steps) |
| TRIG | Trigger/gate input — behaviour is engine-dependent (attack excitation, new note, etc.) |

## Outputs

| Jack | Function |
| --- | --- |
| OUT1 | Main audio output |
| OUT2 | Auxiliary output — content depends on engine |

## Display

The OLED display shows the current bank name and model name, making it easy to identify the active engine at a glance.
