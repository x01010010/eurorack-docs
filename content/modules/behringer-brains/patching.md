# Patch Recipes

## Basic Melodic Voice

Patch a sequencer V/OCT to V/OCT, a gate to TRIG. Select a Bank A analogue model. Patch OUT1 through a VCA and envelope for a classic subtractive-style voice. Adjust HARMONICS for brightness and MORPH for waveform character.

## FM Percussion

Select the two-operator FM engine (Bank A). Patch a short gate to TRIG, patch an envelope to TIMBRE CV to sweep operator ratio dynamically. MORPH controls feedback — push it high for metallic clangs and toms.

## Physical String with Body

Select the string model (Bank C). Patch OUT1 to the signal path and OUT2 to a send/return reverb independently — the resonator body on OUT2 responds to reverb more naturally than a combined signal. Patch velocity CV to TIMBRE to vary pluck hardness.

## Chord Pad

Select the chord engine (Bank B). HARMONICS selects the chord type (major, minor, sus4, etc.). Patch a slow LFO to MORPH to animate the voicing. OUT2 carries a detuned variant — blend for width.

## Sequenced Engine Morphing

Patch a quantised CV from a sequencer to MODEL. Each step triggers a different engine, creating timbral variation that follows the same pitch sequence. Combine with a clock divider on TRIG for rhythmic interest.

## Granular Texture Drone

Select a Bank D granular engine. Leave TRIG unpatched or feed a very slow gate. TIMBRE and MORPH control grain density and size — sweep both with slow LFOs for an evolving pad. Use OUT1 and OUT2 panned wide for stereo spread.
