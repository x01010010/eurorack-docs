# Panel & I/O

## Controls

| Control | Description |
|---|---|
| FREQ | Cutoff frequency |
| RESONANCE | Resonance amount — at maximum the filter self-oscillates |
| LEVEL | Input level attenuator for the audio signal |
| CV | Attenuator for the CV input |
| DRIVE | Switch — engages input overdrive/saturation |
| CHARACTER 1 | Switch — alters resonance behaviour and self-oscillation waveform |
| CHARACTER 2 | Switch — second character flavour; combinations with Character 1 give different tones |

The two character switches are independent toggle switches. Their combination defines the filter's resonance character.

## Inputs

| Jack | Description |
|---|---|
| AUDIO IN | Audio signal input |
| CV | CV input for cutoff modulation (with LEVEL attenuator) |
| V/OCT | 1 V/oct pitch CV input for self-oscillation pitch tracking (character switch positions may affect accuracy) |

## Outputs

| Jack | Description |
|---|---|
| LP | Low-pass output, 2-pole (12 dB/oct) |
| BP | Band-pass output, 1-pole (6 dB/oct) |
| HP | High-pass output, 2-pole (12 dB/oct) |

When self-oscillating: LP = 0°, BP = 90°, HP = 180° sine wave phase relationship.

## Notes

- All three outputs are simultaneously active and can be used at once.
- The V/OCT input allows the self-oscillating filter to track keyboard or sequencer pitch. Character switch combinations with high resonance may reduce V/Oct accuracy.
