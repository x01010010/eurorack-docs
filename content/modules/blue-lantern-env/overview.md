# Overview

**ENV** is an 8 HP digital ADSR envelope generator from Blue Lantern. It covers the full range of envelope tasks — standard one-shot gate response, gated looping for rhythmic modulation, and free-running loop mode that turns the output into a low-frequency oscillator.

Six control knobs give direct hands-on access to every stage: Attack, Decay, Sustain, Release, a master Time scale knob that stretches or compresses all time stages simultaneously, and an output Level knob with a dedicated CV input so the envelope's peak amplitude can itself be voltage-controlled.

Two switches define how the envelope behaves. A linear/exponential toggle sets the shape of the time stages — linear produces an even, technical slope while exponential delivers the natural, snappy feel of most analog envelopes. A three-position mode switch selects Normal (single-pass), Gated (looping only while a gate is held), or Loop (free-running LFO-like loop). A third **Punch Mode** switch adds a sharp initial transient to the attack, reinforcing the front edge of the envelope — useful on percussion and bass patches.

The two simultaneous envelope outputs let you route the same shape to two different destinations without a mult. A separate Gate input and Trigger input provide flexibility: patch a gate for held notes or a trigger for one-shot behavior.

## At a Glance

- **8 HP**, 1.5 inches deep
- **Full ADSR + Time + Level** — six knobs
- **CV input for output level** (with attenuating level knob)
- **Linear / Exponential** response switch
- **Normal / Gated / Loop** mode switch
- **Punch Mode** switch for transient attack snap
- **2× simultaneous envelope outputs**
- **Gate + Trigger inputs**
- **Power**: +21 mA / −13 mA
