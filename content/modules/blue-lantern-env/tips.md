# Tips

## Use the TIME Knob for Macro Length Changes

The TIME knob scales all time stages simultaneously — Attack, Decay, and Release — without touching individual knobs. Set the basic shape of your envelope with the ADSR knobs, then use TIME to make that shape play at different tempos or instrument scales (short for percussion, long for pads). This makes it fast to adapt a saved "feel" to a new context.

## Exponential Mode for Natural-Sounding Envelopes

Linear mode produces mathematically straight ramps. Exponential mode curves the ramps the way the ear expects them — a fast initial movement that slows toward the target. For amplitude envelopes, EXP mode makes attacks feel snappier and releases feel smoother and more natural. Switch to LIN when you need precise, predictable CV shapes for modulation rather than sound.

## Punch Mode Without Full Attack

Punch mode adds a spike at the very start of the attack. Used with a slow attack, you get a sound that has a hard initial click followed by a soft swell — the best of both worlds for bass patches that need articulation but still want a long warm build-in. Try Punch with medium attack and a slow TIME setting.

## Dual Outputs Save Mults

The two simultaneous outputs mean you never need to mult the envelope signal when using it on both amplitude and filter at the same time. Just patch Out 1 and Out 2 to two different destinations. If you need different levels at each destination, add an attenuator after one of the outputs.

## Gated Loop for Accent Effects

In GATE mode, the envelope loops rhythmically — but only while a gate is present. Patch a long gate (e.g. from a clock divider set to long pulse widths) to create a filter or tremolo sweep that only activates on certain beats. Combine with a fast ADSR time for fast twitchy cycling, or slow it down for smooth breathing effects.

## Loop Mode Waveform Shaping

In Loop mode, treat the ADSR controls as waveform shapers: ATTACK = rise time, DECAY = fall from peak, SUSTAIN = plateau level, RELEASE = final fall. Setting ATTACK fast and RELEASE slow produces a sawtooth-up shape. Reversing this (ATTACK slow, RELEASE fast) produces a reverse sawtooth. Equal ATTACK and RELEASE with zero SUSTAIN gives a triangle. Experimenting with SUSTAIN level shapes the flat top of the cycle.

## Audio-Rate Looping for Experimental Tones

With TIME at minimum and LOOP mode engaged, the envelope cycles fast enough to produce audio. Patch the output to a VCA audio input — you will hear a buzzing, harmonically complex tone. Modulate TIME with a slow CV source to sweep through sub-audio and audio rates. Not a traditional oscillator, but a unique texture source.

## CV Input for Dynamics

The CV input controls output level. Feed it from a random voltage source, a sample-and-hold, or a sequencer's accent output to add variation to the envelope intensity — simulating velocity or accent without a keyboard. Adjust the LEVEL knob to set how much the CV input affects the output.
