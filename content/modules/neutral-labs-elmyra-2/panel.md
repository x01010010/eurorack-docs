# Panel & I/O

## Voices

**ELMYRA 2** has four independent voices, each with:

- **Touch pad** — capacitive touch activates the voice's envelope
- **CV input** — gate/trigger input, activates envelope via CV (touch and CV can coexist)
- **V/Oct input** — 1 V/octave pitch CV; compatible with chromatic mode and standard keyboard CV
- **Individual sequencer** — per-voice with up to 128 steps, free-running or synced

## Patch Points (41 total)

The front panel exposes a dense patch bay. Key signals available include:

| Section | Signals |
|---|---|
| Voice CV | 4× CV input (V/oct), 4× gate/trigger input |
| Envelopes | 4× envelope output (from touch or CV) |
| LFOs | 2× LFO output, LFO rate CV input |
| Modulation | Multiple mod target inputs across voices |
| Effects | Delay send/return, delay time CV, delay feedback CV |
| Filter | Filter cutoff CV input, resonance CV input |
| Utilities | Attenuator inputs/outputs, buffered mult, CV summing inputs |
| Audio | Audio input (external signal), audio output (L/R) |

## Preset Card & Component Slots

Jumper inserts along the front panel accept Neutral Labs preset cards (4 included) or bare electronic components (diodes, capacitors) to change the tonal character of the OUCH circuit and oscillator behavior. No patching required — simply insert a card.

## Utilities

**ELMYRA 2** includes an onboard utility section:

- **Dual attenuator / buffered multiple / voltage generator** — bias voltage or process CV before routing
- **CV summing circuit** — combine multiple modulation sources before a target

## Audio I/O

| Jack | Description |
|---|---|
| AUDIO IN | External audio input — processed through the filter, OUCH, and effects chain |
| OUT L / OUT R | Stereo audio output (mono-compatible) |

## Controls

The panel is divided by voice and global sections. Per-voice controls include level, tuning, and sequencer access. Global controls include filter cutoff, resonance, filter mode select, delay time/feedback/mix, reverb mix, OUCH drive, and LFO rate/depth.
