# Patch Recipes

## Four-Voice Polyphonic VCA

Patch four VCO outputs to IN 1–4. Patch four independent ADSR envelopes to CV 1–4. Each voice gets its own amplitude envelope. Take MIX to the output or further processing. Full four-voice polyphonic amplitude control.

## Single Envelope for All Voices

Patch one ADSR to CV 1. Leave CV 2–4 unpatched (they normalise from channel 1). All four channels follow the same envelope. Useful for unison or layered voices that should always play together.

## Voltage-Controlled Crossfade

Patch two audio signals to IN 1 and IN 2. Patch a single LFO to CV 1 (channels 1+2 normalise from it). Set CURVE to LIN. As the LFO rises, channel 1 opens and channel 2 responds to the same CV — but patching the same CV inverted to channel 2 directly creates a true crossfade. Use an attenuverter to invert. Take MIX.

## Layered Drum Mix with CV Accents

Patch four drum voices to IN 1–4. Set LEVEL knobs for a static mix balance. Patch an accent trigger to an envelope → CV 1 on the kick channel. Each accent trigger briefly opens the kick higher, creating dynamic accents.

## CV Modulation Mixing

Patch four modulation CVs (LFOs, envelopes, S&H) to IN 1–4. Take MIX to a single VCF cutoff. Use LEVEL knobs to weight each modulation source. A CV mixer with per-channel level control, all in one module.

## Tremolo on Audio

Patch an audio signal to IN 1. Patch an LFO to CV 1. Set CURVE to EXP. The LFO amplitude-modulates the audio, producing tremolo. Adjust CV ATT for tremolo depth. Use the LFO rate for slow tremolo or push into audio rate for AM/ring-mod effects.
