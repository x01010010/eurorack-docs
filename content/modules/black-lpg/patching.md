# Patch Recipes

## Classic LPG Bongo

Set mode to VCF+VCA. Patch an oscillator into IN. Send a trigger from a sequencer or clock into TRIG. Set DECAY short (9–10 o'clock). Adjust OFFSET to taste — lower settings give darker, woodier plucks. The vactrol response naturally produces the organic attack and decay characteristic of LPG percussion.

## Self-Oscillating Drone

Set mode to VCF. Turn RESONANCE fully clockwise and engage OVERDRIVE. The filter will self-oscillate, producing a sine-like tone without any audio input. Modulate the CV input with a slow LFO to create pitch movement. The EG OUT can modulate another module in sync.

## Envelope Follower Substitute

Patch a rhythmic audio signal into IN and use the TRIG input with a clock synced to your sequence. The EG OUT provides a decay envelope on each trigger — patch it into another module's CV input (filter, panning, waveshaper) to create rhythmically linked modulation without a dedicated envelope follower.

## Ping Filter

In VCF mode with moderate resonance, send a short trigger to TRIG with nothing patched to IN. The envelope "pings" the resonant filter, producing a tuned percussion sound. Adjust OFFSET to set the pitch. Add overdrive for more aggressive attack.

## Dynamic Timbre Sequencing

Patch a sequencer CV into DECAY CV to vary the envelope length per step. Shorter decays produce tight clicks; longer decays let the filter open further. Combined with VCF+VCA mode, each step of your sequence gets a different timbral shape.

## Sidechain-Style Ducking

In VCA mode, patch your pad or drone into IN. Send a kick drum trigger into TRIG. Use EG OUT inverted (via an attenuverter) into the CV input. Each kick trigger dips the amplitude of the pad, creating a pumping sidechain effect.
