# Modes

Black LPG has three operating modes selectable via the front-panel MODE control or the MODE CV input.

## VCF Mode

Only the low-pass filter is active. The VCA is bypassed — audio passes through at unity gain.

- **OFFSET** controls the filter cutoff frequency.
- **RESONANCE** and **OVERDRIVE** are fully active.
- Use this when you already have a separate VCA or envelope controlling amplitude elsewhere in your chain.

## VCF+VCA Mode

Both the low-pass filter and the VCA are active in series. The CV/envelope controls both cutoff and amplitude simultaneously.

- **OFFSET** sets both the base cutoff and base amplitude.
- This is the classic LPG behavior: as the signal gets quieter, it also gets darker — mimicking the physics of acoustic instruments.
- Best for plucks, bongos, and any "struck object" sound.

## VCA Mode

Only the voltage-controlled amplifier is active. The filter is bypassed.

- **OFFSET** controls the base amplitude level.
- **RESONANCE** has no effect in this mode.
- The vactrol response still provides smooth, organic amplitude shaping even without the filter.
- Use this as a standalone VCA with built-in envelope when you want transparent amplitude control.

## Mode CV

The MODE CV input allows voltage-controlled switching between modes. This enables dynamic mode changes within a patch — for example, morphing from filtered plucks to clean VCA passes on alternate triggers using a sequenced CV.
