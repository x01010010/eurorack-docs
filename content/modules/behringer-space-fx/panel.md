# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| PROGRAM | Knob | Selects one of 32 effect algorithms |
| PARAM 1 | Knob | Primary parameter for current algorithm (e.g., reverb time, delay time) |
| PARAM 2 | Knob | Secondary parameter for current algorithm (e.g., damping, feedback) |
| TAP / PARAM 3 | Button | Tap tempo for time-based effects; also accesses a third parameter on some algorithms |
| LEVEL | Switch | Sets output level and wet/dry balance |
| DISPLAY | 8-character LED display | Shows current program name and parameter values |

## Inputs

| Jack | Function |
| --- | --- |
| FX SEND (IN) | Mono audio input |

## Outputs

| Jack | Function |
| --- | --- |
| OUT L | Stereo left output |
| OUT R | Stereo right output |
