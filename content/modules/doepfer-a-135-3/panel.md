# Panel & I/O

## Controls (per channel, ×4)

| Control | Description |
|---|---|
| L1–L4 | Level knob — manually sets VCA gain for both L and R of that channel. Small rubberized knob. Full clockwise = maximum gain (~0.5). |
| CV 1–4 attenuator | Trims the incoming CV range. Plastic shaft with white marker line; no knob. Attenuator full clockwise allows 0–5 V to sweep the full VCA range. Higher CV voltages can be scaled down with the attenuator. |

## Inputs (per channel, ×4)

| Jack | Description |
|---|---|
| InL 1–4 | Left audio/CV input for channels 1–4. |
| InR 1–4 | Right audio/CV input for channels 1–4. |
| CV 1–4 | Control voltage input for channel amplitude. 0–5 V for full range (attenuator full CW). Signals up to 15 Vpp can be processed without distortion. |

## Outputs

| Jack | Description |
|---|---|
| Out L | Summed stereo left bus output. |
| Out R | Summed stereo right bus output. |

## Panel Notes

- Signal inputs have no onboard attenuators. The module handles up to 15 Vpp clean; use an external attenuator (e.g., Doepfer A-183-1) for hotter signals.
- All inputs and outputs are DC coupled and non-inverting.
- CV attenuators have no knob — adjust with a screwdriver or finger on the plastic shaft.
- Maximum VCA gain is ~0.5 by default. Two resistors (R27, R31) can be changed to 100 kΩ to raise this to unity gain — see the factory jumper document.
- Chaining jumpers on the PCB configure a unit as master or slave for expanding to 8+ channels.
