# Overview

**A-135-3** is an 8 HP voltage-controlled stereo mixer from Doepfer, built around eight linear CEM3360/AS3360 VCA circuits — two per channel, one for L and one for R. Four stereo input channels each have a manual level knob and a CV input with an associated attenuator, giving independent voltage control over every channel's amplitude in the stereo bus.

The design is deliberately minimal: no panning, no sends, no mutes — just four channels of clean, linear VCA-based gain that sum to a stereo output. Because all signal paths are DC coupled and non-inverting, the module works equally well as an audio mixer and as a CV combiner. Two or more units can be linked in a master/slave configuration using internal jumpers to extend the channel count beyond four.

The maximum gain per channel is approximately 0.5 (−6 dB) by default, chosen to prevent clipping when all four channels are running at full level. This can be raised to unity gain by replacing two resistors per the factory documentation.
