# Patching

## Classic Fuzz Lead

Aggressive, heavily distorted lead synth tone with treble boost.

```
VCO Out → PUNK IN
HIGH EQ: boost
FUZZ GAIN: up (MOSFET mode)
FET GAIN: down
BLEND: full wet (or slightly backed off for dynamics)
PUNK OUT → VCF In
```

## Fattening a Drum Bus

Add cohesive grit and presence to a mixed drum submix without destroying clarity.

```
Drum Mix → PUNK IN
INPUT LEVEL: moderate
FET GAIN: ~50%
FUZZ GAIN: down
BLEND: ~30% (mostly clean, touch of grit)
LOW EQ: slight boost for weight
HIGH EQ: slight cut to tame transients
PUNK OUT → Main Mix
```

## Saturated Bass

Thick, warm, powerful low-end with preamp fatness and fuzz attitude.

```
Bass VCO (Square/Saw) → PUNK IN
FET GAIN: ~60%
FUZZ GAIN: ~70% (Diode mode for warmth)
BLEND: full wet
LOW EQ: subtle boost
PUNK OUT → VCF In
```

## Gritty Metallic Sounds

Use diode fuzz with HIGH EQ boost for industrial, aggressive tones.

```
Any signal source → PUNK IN
FET GAIN: high
FUZZ GAIN: high (Diode mode)
HIGH EQ: high boost
LOW EQ: cut or flat
PUNK OUT → reverb/delay
```

## Transient Preservation (Parallel Style)

Blend in just enough distortion to add presence without losing the original attack.

```
Drum (kick, snare, tom) → PUNK IN
FET GAIN: ~40%
FUZZ GAIN: down
BLEND: 20–40% (mostly dry)
PUNK OUT → Mix with original drum signal
```

## Ambient Texture Distortion

Subtle, musical coloration for pads and atmospheric sounds.

```
VCO/Noise → PUNK IN
INPUT LEVEL: low
FET GAIN: ~30%
FUZZ GAIN: down
BLEND: ~15% (mostly clean)
LOW EQ: subtle
HIGH EQ: subtle
PUNK OUT → Reverb/Delay → Speaker
```

## Multi-Layer Distortion with Send/Return

Use PUNK in a send/return loop for controllable parallel distortion on a larger mix.

```
Main Mix → Send
Send → PUNK IN
BLEND: full wet (since you're already in a send return)
FET GAIN + FUZZ GAIN: set to taste
PUNK OUT → Return Fader
```
Dial the return fader to blend in as much or as little distortion as needed.

## Chaining with Other Effects

**PUNK** works well before or after other processors:

- **Before compression** — let PUNK distort, then compress for cohesion
- **After compression** — compress for dynamics control, then add PUNK distortion for color
- **Into reverb** — distort and then mangle with reverb tail for atmospheric effects
- **With EQ** — use PUNK's EQ to shape before feeding a VCF or other effect

The low 35 mm depth and minimal power make PUNK a practical chain-building module.
