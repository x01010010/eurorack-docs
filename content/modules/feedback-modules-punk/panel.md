# Panel & I/O

## Controls

| Control | Description |
|---|---|
| **INPUT LEVEL** | Sets the input gain before distortion. Affects how hard both distortion stages are driven. |
| **FET GAIN** | Controls the saturation of the 80s-inspired FET preamp circuit. |
| **FUZZ GAIN** | Controls the saturation of the guitar pedal fuzz distortion stage. |
| **DIODE/MOSFET** | Toggle switch — selects the character of the fuzz distortion (Diode for warmth, MOSFET for aggression). |
| **LOW EQ** | Baxandall shelving EQ for bass frequencies. Boost or cut below ~200 Hz. |
| **HIGH EQ** | Baxandall shelving EQ for treble frequencies. Boost or cut above ~2 kHz. |
| **BLEND** | Mixes between clean dry signal (fully CCW) and fully distorted signal (fully CW). |

## Inputs

| Jack | Description |
|---|---|
| IN | Audio input. Connect any audio source: drum bus, bass, lead synth, drum hits, etc. |

## Outputs

| Jack | Description |
|---|---|
| OUT | Processed audio output (distorted, EQ'd, blended). |

## Signal Flow

INPUT → INPUT LEVEL → (FET GAIN + FUZZ GAIN + DIODE/MOSFET switch) → LOW EQ + HIGH EQ → BLEND (clean/distorted mix) → OUT

## Distortion Characters

**FET Preamp** — warm, saturated, smooth. Based on vintage 1980s mixing console and cassette recorder preamp circuits. Best for fattening, adding harmonic grit without harshness.

**Fuzz (Diode mode)** — classic guitar fuzz character, warm, gritty, sustaining. Traditional character.

**Fuzz (MOSFET mode)** — more aggressive, brighter, more responsive to dynamics. Steeper gain curve, more "punk" attitude (hence the module name).

Both fuzz and preamp can be used together. Dial in the blend to taste: all preamp for fattening, all fuzz for aggression, or split the difference for complex textures.
