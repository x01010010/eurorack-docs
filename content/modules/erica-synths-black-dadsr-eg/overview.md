# Overview

**BLACK DADSR EG** is a fully analogue, exponential envelope generator from Erica Synths with a Delay-Attack-Decay-Sustain-Release (DADSR) structure. The D stage is an independent gate delay section that holds the incoming gate before the ADSR portion fires. This makes it possible to design delayed modulation events, staggered voices, and rhythmically offset envelope responses without any external utilities.

Every stage of the envelope — Attack, Decay, Sustain, and Release — has both a manual knob and a CV input with attenuator, giving fine voltage control over the entire envelope shape. The envelope is re-triggerable: a new gate during the envelope's active phase resets and restarts from Attack.

A looping function converts the envelope into a free-running LFO-like signal, cycling through its ADSR stages continuously. Combined with the CV inputs this makes it a highly shapeable complex modulation source. A manual gate button allows hands-on triggering without a gate source.

## At a Glance

- **10 HP**, 30 mm deep
- **DADSR**: independent gate delay + Attack / Decay / Sustain / Release
- **Exponential response**, fully analogue
- **CV control over every stage** with attenuators
- **Gate delay** up to ~0.7 s
- **Attack**: 0–2.5 s (4.4 s with max CV)
- **Decay**: 0–5 s (12 s with max CV)
- **Release**: 0–4.5 s (10 s with max CV)
- **Looping function** — sustained cycling mode
- **Re-triggerable**
- **Manual gate button**
- **Power**: 42 mA (+12 V) / 58 mA (−12 V)
