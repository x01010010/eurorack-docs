# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| JOYSTICK | 4-axis joystick | Sets the vector blend point between the four oscillators |
| OLED DISPLAY | Display | Shows vector position, waveform names for each oscillator |
| WAVEFORM A–D | Knobs/encoders ×4 | Selects the waveform (of 128) for each oscillator |

## Inputs

| Jack | Function |
| --- | --- |
| V/OCT | 1V/octave pitch CV — controls all four oscillators together |
| GATE | Gate input for voice triggering |
| FM | Linear FM input |
| X IN | CV control of X-axis vector position (overrides joystick X) |
| Y IN | CV control of Y-axis vector position (overrides joystick Y) |

## Outputs

| Jack | Function |
| --- | --- |
| MIX | Full vector-blended mix of all four oscillators |
| X | X-axis blend output (oscillators A+B vs C+D) |
| Y | Y-axis blend output (oscillators A+C vs B+D) |
