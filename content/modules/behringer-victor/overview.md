# Overview

Victor is a quad vector VCO based on the Sequential Prophet VS architecture. It contains four independent digital oscillators, each with 128 waveforms, that can be blended using a 4-axis joystick for vector synthesis. Waveform selection and vector position are voltage-controllable, enabling animated, evolving timbres from simple CV sources.

An OLED display shows the current vector position and selected waveforms. Three outputs provide the mixed vector signal (MIX), the X-axis blend (X), and the Y-axis blend (Y) independently.
