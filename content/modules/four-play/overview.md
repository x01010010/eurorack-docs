# Overview

Four Play is a quad voltage-controlled amplifier based on the Mutable Instruments Quad VCA architecture. It provides four independent VCA channels, each with a LEVEL knob, CV attenuator, and response curve switch. A cascaded CV input chain allows a single CV to control all four channels simultaneously, and a MIX output sums all four channels for use as a voltage-controlled mixer.

Four Play handles both audio and CV signals, making it suitable for voice-level VCA duties, modulation mixing, and dynamic voltage processing.
