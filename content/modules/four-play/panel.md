# Panel & I/O

## Controls (per channel, ×4)

| Control | Type | Function |
| --- | --- | --- |
| LEVEL | Knob | Sets the base gain of the channel (0 to full) |
| CV ATT | Knob | Attenuates the CV input for this channel |
| BOOST | Switch | Enables a gain boost for hot output levels |
| CURVE | Switch | Selects exponential or linear VCA response curve |

## Inputs (per channel)

| Jack | Function |
| --- | --- |
| IN | Signal input (audio or CV) |
| CV | CV input for gain control; normalled from channel above if unpatched |

## Outputs (per channel)

| Jack | Function |
| --- | --- |
| OUT | Amplified output for the channel |

## Summed Output

| Jack | Function |
| --- | --- |
| MIX | Sum of all four channel outputs |
