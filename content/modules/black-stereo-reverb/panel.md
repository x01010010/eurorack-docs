# Panel & I/O

## Controls

| Control | Description |
|---|---|
| SOURCE | Selects the reverb delay line type: Tape, BBD, or dirty BBD |
| MODE | Selects room size mode: Room, Hall, or Cathedral |
| SIZE | Controls the reverb room size within the selected mode |
| FEEDBACK | Controls the reverb tail length / decay time |
| TONE | Shapes the frequency character of the feedback (darker to brighter) |
| SPIN | Controls the stereo spread and rotation of the reverb image |
| DRY/WET | Blends dry input signal with wet reverb output |
| PRESET | Selects one of 10 preset slots. Hold to save; short press to recall |
| FREEZE | Momentary button — holds the current reverb tail indefinitely while pressed |

## Inputs

| Jack | Description |
|---|---|
| IN L | Left audio input |
| IN R | Right audio input (normalled to IN L for mono-to-stereo operation) |
| SIZE CV | CV input for room size control |
| FEEDBACK CV | CV input for feedback/decay control |
| TONE CV | CV input for tone control |
| DRY/WET CV | CV input for dry/wet mix control |
| PRESET CV | CV input for preset selection |
| FREEZE CV | Gate input — high gate activates Freeze |

CV inputs accept ±5 V for full parameter range.

## Outputs

| Jack | Description |
|---|---|
| OUT L | Left stereo output |
| OUT R | Right stereo output |

## Notes

- With only IN L patched, the signal is processed as mono and spread to both outputs via the stereo spin algorithm.
- Maximum audio input level before clipping: 16 VPTP.
- Preset save: hold the PRESET knob/button until the LED confirms the save. Recall: short press to step through presets.
