# Patch Recipes

## Ambient Pad Machine

Patch a sustained chord or drone into IN 1. Set routing to VCF → FX → Distortion. Set REVERB to 3 o'clock, TAIL long. Engage reverse reverb (ROUTING + HOLD). Add a slow LFO into the VCF CV input for a gentle filter swell. The result is a slowly evolving, reverse-washed ambient texture.

## Karplus-Strong Plucked Strings

Set delay TIME very short (below 50 ms). Turn REPEATS to near maximum. Patch a 1V/oct sequence into the TIME CV input. Send short trigger pulses into IN 1. Each trigger excites the delay as a resonator, producing pitched plucked-string sounds tracking your sequence. Adjust TIME/DIV to tune the base pitch.

## Sidechain Pump

Patch your bass/pad into IN 1. Patch a kick trigger into SIDECHAIN TRIG. Set SIDECHAIN knob to taste for release time. Hold ROUTING and turn SIDECHAIN to set ducking depth. The pad ducks on every kick trigger and recovers smoothly — classic pumping sidechain effect. Patch ENV OUT to other modules for synced modulation.

## Drum Bus Glue

Patch a mixed drum signal into IN 1. Set compressor to full CW (drum preset). Set VOLUME/DRIVE past 3 o'clock for subtle output saturation. Routing: Distortion → VCF → FX with distortion minimal and reverb off gives parallel-compression-like glue with a touch of harmonic saturation.

## Comb Filter Drone

Switch filter to comb mode (BPF/COMB × 2). Patch a noise source or drone into IN 1. Set RESONANCE high. Modulate FREQUENCY with a slow LFO or a sequence. The comb filter extracts pitched harmonics from the noise, producing a metallic evolving drone. Add reverb for depth.

## Rhythmic Tap Delay with Clock Sync

Patch your system clock into CLOCK IN. Turn TIME/DIV to set the division (12 o'clock = clock tempo, CCW = divisions). Set stereo tap pattern to LRRL for a wide, rhythmically offset stereo spread. REPEATS around 10 o'clock for defined echoes that don't wash out.
