# Overview

**OUT** is a 2 HP stereo line output and headphone module from 2hp. It bridges the gap between modular-level signals and the outside world — a line-level interface, a recording chain, a headphone amp — all in the smallest possible footprint.

Stereo modular-level inputs feed both a balanced stereo line output and a 3.5 mm headphone jack. The two output sections are independent with separate volume controls, so headphone monitoring level and line output level can be set differently without affecting one another. Left and right LED VU meters on the inputs give a quick visual read on signal level before it hits the outputs.

The analog signal path is designed for a low noise floor, making **OUT** appropriate for direct recording and high-fidelity monitoring rather than just casual listening.
