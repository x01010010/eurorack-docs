# Panel & I/O

## Controls

| Control | Description |
|---|---|
| LINE VOL | Volume knob for the stereo line output. Controls the level sent to the line-out jack independently of the headphone output. |
| HP VOL | Volume knob for the headphone output. Controls headphone listening level independently of line output. |

## Inputs

| Jack | Description |
|---|---|
| L IN | Left channel modular-level audio input. |
| R IN | Right channel modular-level audio input. |

## Outputs

| Jack | Description |
|---|---|
| LINE OUT | Stereo 3.5 mm line output. Connect to an audio interface, mixer, or external effects unit. |
| HP | 3.5 mm headphone output. Drives standard headphones for monitoring. |

## Indicators

| Indicator | Description |
|---|---|
| L LED | VU meter for the left input channel. Brightness indicates signal level. |
| R LED | VU meter for the right input channel. |

## Panel Notes

- Plugging only into L IN will not auto-sum to both channels — patch both L and R for stereo output.
- LINE OUT and HP output simultaneously; the two volume knobs are fully independent.
