# Patch Recipes

## Basic Monitoring Setup

The simplest use: patch your final stereo mix into L IN and R IN, set LINE VOL and HP VOL to taste, and connect headphones to HP and a recording interface to LINE OUT.

```
final mix L → L IN
final mix R → R IN
HP → headphones
LINE OUT → audio interface input L/R
```

## Headphones While Recording

Because LINE OUT and HP operate simultaneously with independent volumes, you can monitor at a comfortable headphone level while recording at a consistent line level without one affecting the other.

## Sending to External Effects

Route LINE OUT to an external effect (reverb pedal, delay unit, hardware compressor) and return the processed signal back into your modular for further processing. **OUT** serves as the send stage.

```
modular mix L/R → L IN / R IN
LINE OUT → external effect input
effect output → modular input module
```

## Mono Sum to a Single Destination

Patch the same signal into both L IN and R IN. The line output and headphone output both carry that signal on both channels, effectively driving a mono signal to a stereo destination at consistent level on each side.
