# Overview

**PROGRAMM** is a 12-channel MIDI and CV sequencer designed to work at the heart of hybrid and modular setups. It bridges the gap between MIDI controllers and modular CV sequencers, offering intuitive real-time pattern control with flexible channel routing to both CV and MIDI destinations.

## Key Features

- **12 Channels** with independent pattern sequencing and output routing
- **MIDI and CV Outputs** — route each channel to MIDI notes, CC, or CV gate/note
- **Clock Generator** with internal or external sync
- **Random Mode** for generative sequences
- **USB and MIDI I/O** for host control and hardware synth integration
- **Compact 40 HP** form factor ideal for studio or live performance

## Workflow

**PROGRAMM** excels in polyrhythmic and generative contexts. Load patterns into individual channels, tap in real-time, sync to external clock or MIDI, and modulate via CV inputs. The MIDI/CV duality makes it as comfortable controlling external gear as it is sequencing a Eurorack voice.
