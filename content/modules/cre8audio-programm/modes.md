# Modes

**PROGRAMM** operates in several complementary modes, selectable from the front panel or via MIDI control.

## Sequencer Modes

### Pattern Edit
Tap steps on the channel grid to draw melodies or rhythm patterns. Each step can have independent gate length, velocity, and timing offset.

### Chord Mode
Stack notes in a single step to create chords, then transpose the entire pattern using the octave control.

### Random Generative
Activate random mode to generate evolving sequences. Seed and constraining parameters control the degree of chaos.

## MIDI/CV Output Routing

Each of the 12 channels can be independently routed to:
- **MIDI Note Out** — velocity-sensitive
- **MIDI CC Out** — continuous controller messages
- **CV Gate + Pitch** — simultaneous gate and 1V/octave pitch CV to Eurorack modules

## Sync Modes

- **Internal Clock** — tempo set via tempo knob
- **External Sync** — sync to Eurorack clock or MIDI clock
- **MIDI Beat Clock** — clock generated from host DAW or MIDI controller
