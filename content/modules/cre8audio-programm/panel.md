# Panel

The **PROGRAMM** front panel is dominated by a 12-channel grid display and control matrix. The top third houses the channel selector buttons; the middle section contains per-channel knobs and LEDs; the bottom third has master playback controls, MIDI/sync I/O indicators, and USB port.

## Layout

- **Channel Buttons** (left column) — select active channel for editing
- **Pattern Grid Display** — shows sequence steps for the active channel
- **Per-Channel Controls**:
  - Knob for parameter adjustment (rate, swing, octave, CC value, etc.)
  - LED status indicator
- **Master Controls** (right column):
  - Play/Pause, Tempo, Sync mode
  - Pattern select buttons
  - MIDI channel/output routing
- **I/O** (bottom edge):
  - USB port (power + MIDI/data)
  - MIDI IN / OUT jacks
  - Sync IN (clock)

## Visual Feedback

Bright LEDs on each channel light as notes trigger. The main display cycles through pattern steps in real-time. Active MIDI/clock status is shown via dedicated indicator LEDs.
