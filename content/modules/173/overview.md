# Overview

The 173 is a dual-purpose utility module based on the Roland System 100M 173 Quad Gate/Multiples. One section provides four independent signal buffers (multiples), each with one input and four outputs, for distributing a single signal to multiple destinations. The second section provides gate/trigger logic functions useful for rhythmic combination and distribution.

It is a "patchbay extender" for a modular system — any time a CV, audio, or gate signal needs to reach more than one destination, the 173 provides buffered multiple outputs without signal degradation.
