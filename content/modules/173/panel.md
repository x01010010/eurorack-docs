# Panel & I/O

## Multiples Section

Six independent 1-in / 4-out signal buffers. Any signal type (CV, audio, gate, clock) can be connected to the input and distributed to up to four destinations simultaneously.

| Jack | Function |
| --- | --- |
| MULT A IN | Input for multiple A |
| MULT A OUT 1–4 | Four buffered outputs from input A |
| MULT B IN | Input for multiple B |
| MULT B OUT 1–4 | Four buffered outputs from input B |
| (×4 more pairs) | Four additional multiple pairs |

## Gate Section

Four gate/trigger processing channels.

| Jack/Control | Function |
| --- | --- |
| GATE IN A, B | Gate/trigger inputs |
| AND OUT | Goes high only when both A and B are high simultaneously |
| OR OUT | Goes high when either A or B is high |
| NOT A OUT | Inverted gate of input A |
| NOT B OUT | Inverted gate of input B |

*Note: exact jack labelling varies by hardware revision — consult the panel silkscreen.*
