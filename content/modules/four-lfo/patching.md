# Patch Recipes

## Stereo Chorus/Vibrato

Set channels 1 and 2 to Quadrature mode. Patch CH1 SIN to VCO 1 FM and CH2 SIN (90° offset) to VCO 2 FM. The two VCOs modulate in phase quadrature — when panned L/R, this creates a wide, natural-sounding stereo vibrato.

## Rhythmic Polyrhythm Modulation

Set channel 1 to Free, channels 2–4 to Divide. Set division ratios of 2, 3, and 4. You now have four rhythmically related LFOs at 1×, ½×, ⅓×, and ¼× speed. Patch them to filter, VCA, and pitch for layered rhythmic modulation.

## Slow Evolving Pad Texture

Set all four channels to Free at very different (slow) rates. Patch SIN outputs to filter cutoff, pan, reverb mix, and pitch on a sustained pad. With four LFOs at incommensurable rates, the texture evolves for a very long time before (approximately) repeating.

## Vector Joystick Substitute

Set channels 1 and 2 to Quadrature. Patch CH1 SIN and CH2 SIN (0° and 90°) to X and Y inputs on a vector/wavetable oscillator. The LFOs trace a circular path through the sound space — a budget substitute for a joystick CV controller.

## Gate-Reset Sync

Patch a master clock or sequencer gate to all four RESET inputs. Each LFO restarts its cycle on every clock pulse, keeping modulation phase-locked to the rhythmic grid regardless of rate differences.

## Phase Modulation Texture

Set all four channels to Phase mode. Use the four RATE faders to set arbitrary phase offsets (0°, 45°, 135°, 220°, for example). Patch all four SIN outputs to separate destinations — the subtle phase relationships create complex, quasi-random-sounding modulation from mathematically simple sources.
