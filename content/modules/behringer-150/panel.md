# Panel & I/O

## Ring Modulator Section

| Jack/Control | Type | Function |
| --- | --- | --- |
| X IN | Input | Carrier signal input |
| Y IN | Input | Modulator signal input |
| RING MOD OUT | Output | X × Y ring modulated output |

## Noise Section

| Jack | Type | Function |
| --- | --- | --- |
| WHITE 1 | Output | White noise output A |
| WHITE 2 | Output | White noise output B |
| PINK 1 | Output | Pink noise output A |
| PINK 2 | Output | Pink noise output B |

## Sample & Hold Section

| Jack/Control | Type | Function |
| --- | --- | --- |
| S&H IN | Input | Signal to be sampled |
| TRIG IN | Input | External trigger input — each rising edge takes a sample |
| S&H OUT | Output | Held output voltage |
| RATE | Knob | Internal S&H clock rate (used when no external trigger patched) |

## LFO Section

| Jack/Control | Type | Function |
| --- | --- | --- |
| RATE | Knob | LFO frequency |
| SINE OUT | Output | Sine wave |
| TRIANGLE OUT | Output | Triangle wave |
| SAW OUT | Output | Sawtooth wave |
| RAMP OUT | Output | Reverse sawtooth (ramp up) |
| SQUARE OUT | Output | Square wave |
