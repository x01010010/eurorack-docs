# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| FREQ | Knob | Base resonator frequency (root pitch of the resonator) |
| STRUCTURE | Knob | Controls the spread and character of the mode series — from harmonic to inharmonic |
| BRIGHTNESS | Knob | High-frequency content — damping of upper partials |
| DAMPING | Knob | Decay time of the resonator |
| POSITION | Knob | Excitation position along the resonating object — affects harmonic balance |

## Inputs

| Jack | Function |
| --- | --- |
| V/OCT | 1V/octave pitch CV — transposes the resonator frequency |
| STRUM | Trigger/gate input — excites the resonator with an internal impulse (no external audio needed) |
| IN | External audio input — the resonator processes and colours the incoming signal |
| STRUCTURE CV | CV over STRUCTURE parameter |
| BRIGHTNESS CV | CV over BRIGHTNESS parameter |
| DAMPING CV | CV over DAMPING parameter |
| POSITION CV | CV over POSITION parameter |

## Outputs

| Jack | Function |
| --- | --- |
| OUT | Main resonator output (mono or stereo sum) |
| ODD | Odd harmonics of the resonator |
| EVEN | Even harmonics of the resonator |
