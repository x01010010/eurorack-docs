# Overview

Four LFO is a quad low-frequency oscillator based on the Xaoc Devices Batumi architecture. It provides four independent LFO channels, each with three simultaneous outputs: a sine wave, a user-assignable wave, and a square wave. Each channel has an LED fader for rate control and a mode switch for selecting the channel's relationship to the other LFOs.

Per-channel modes include free-running, quadrature (90° offset), phase-offset (arbitrary offset relative to channel 1), and divide (integer division of channel 1's rate). This makes Four LFO a dense, flexible modulation source for complex, interrelated modulation patterns.
