# Panel & I/O

## Controls (per channel, ×4)

| Control | Type | Function |
| --- | --- | --- |
| RATE fader | LED fader | Sets LFO rate; LED brightens at the fader position |
| MODE switch | 4-position switch | Selects channel operating mode (Free, Quad, Phase, Divide) |

## Inputs (per channel)

| Jack | Function |
| --- | --- |
| RESET | Rising edge resets LFO phase to zero |
| RATE CV | CV input scales LFO rate; combined with fader position |

## Outputs (per channel)

| Jack | Function |
| --- | --- |
| SIN | Sine wave output |
| ASN | Assignable wave output — waveform selected per channel in mode config |
| SQR | Square wave output |

All outputs are bipolar (±5 V typical).
