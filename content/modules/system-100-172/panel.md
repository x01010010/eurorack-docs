# Panel & I/O

The panel divides into four stacked sections.

## Phase shifter section

| Control | Function |
| --- | --- |
| EFFECT ON/OFF | Engages the phaser |
| MOD INTENSITY | How much the LFO or external CV modulates SHIFT FREQ |
| SHIFT FREQ | Phase shift center frequency, 200 Hz to 8 kHz |
| RESONANCE | Feedback amount through the all-pass network |
| CV IN | External CV (0–10 V) replaces the internal LFO |
| SIG IN | Audio in, AC coupled |
| SIG OUT | Phase-shifted audio out |

## Delay section

| Control | Function |
| --- | --- |
| EFFECT ON/OFF | Engages the BBD delay |
| MOD INTENSITY | LFO/CV depth into delay time |
| DELAY TIME | 1 ms to 7 ms |
| RESONANCE | Feedback into the BBD line |
| CV IN | External CV (0–10 V) for delay time |
| SIG IN | Audio in, AC coupled |
| SIG OUT | Delayed audio out (wet only) |

To get a chorus/flanger sound you must mix the delay's SIG OUT with the dry signal externally — the 172's delay output is wet-only.

## LFO section

| Control | Function |
| --- | --- |
| FREQUENCY | LFO speed, 0.02 Hz to 10 Hz |
| LFO OUT (×2) | Triangle output (upper) and inverted triangle (lower) |

The LFO is normalled into both the phase shifter CV IN and delay CV IN. Patch a cable into either CV IN to break the normal and use external modulation.

## Gate delay section

| Control | Function |
| --- | --- |
| THRESHOLD | Voltage required to trigger the gate (0.3 V to 5 V) |
| GATE IN | Incoming gate or trigger |
| DELAY TIME | Time before output gate fires (0.3 ms to 6 s) |
| GATE TIME | Length of the output gate (0.3 ms to 6 s) |
| GATE OUT | +5 V output gate |

Use this section to convert short triggers into longer gates, or to delay a gate by a fixed amount — handy for offset envelopes, polyrhythms, or syncopation.
