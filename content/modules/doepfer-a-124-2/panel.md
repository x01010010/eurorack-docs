# Panel & I/O

## Controls

| Control | Description |
|---|---|
| FREQ | Cutoff frequency |
| RESONANCE | Resonance amount (no self-oscillation) |
| MIX | Blends LP and HP signal at the Mix output: fully CCW = LP only, center = Notch, fully CW = HP only |
| INPUT LEVEL | Attenuator for the audio input signal (small trimmer-style control) |
| CV1 | Attenuator for CV input 1 (full-size knob) |
| CV2 | Attenuator for CV input 2 (small trimmer-style control) |

The Mix and Input Level controls use smaller trimmers than the A-124/A-124 SE due to the 4 HP panel width.

## Inputs

| Jack | Description |
|---|---|
| AUDIO IN | Audio signal input |
| CV 1 | CV input 1 for cutoff modulation |
| CV 2 | CV input 2 for cutoff modulation |

## Outputs

| Jack | Description |
|---|---|
| BP OUT | Bandpass output |
| MIX OUT | LP/HP mix output — character set by MIX knob (LP / Notch / HP) |
| LP/HP OUT | Dedicated LP or HP output, selected by internal jumper |

## Jumper Setting

An internal jumper on the PCB determines whether the third output (LP/HP OUT) carries the lowpass or highpass signal. Set once at installation; not a real-time control.

## Notes

- All three outputs can be used simultaneously.
- The Notch position on MIX OUT is sensitive — small deviations from center lean toward LP or HP. Use it as a starting point and tweak to taste.
