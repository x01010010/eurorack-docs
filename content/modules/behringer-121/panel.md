# Panel & I/O

The 121 has two identical filter sections. Controls and jacks are mirrored for each.

## Controls (per filter)

| Control | Type | Function |
| --- | --- | --- |
| CUTOFF | Knob | Filter cutoff frequency |
| RESONANCE | Knob | Resonance / Q — self-oscillates at maximum |
| SIG 1/2/3 LEVEL | Knobs ×3 | Individual input level attenuators for each of the three signal inputs |
| MOD 1/2/3 LEVEL | Knobs ×3 | Attenuators for each of the three modulation CV inputs |
| HP/LP | Switch | Switches filter between high-pass and low-pass mode |

## Inputs (per filter)

| Jack | Function |
| --- | --- |
| SIG 1, 2, 3 | Three independent audio inputs (mixed internally) |
| MOD 1, 2, 3 | Three CV inputs for cutoff modulation (summed) |
| V/OCT | 1V/octave pitch-tracking input for keyboard tracking |

## Outputs (per filter)

| Jack | Function |
| --- | --- |
| OUT | Filtered audio output |
