# Panel & I/O

The panel is 3 HP wide. Controls and jacks are stacked vertically.

## Controls

| Control | Description |
|---|---|
| EFFECT | Rotary selector — chooses one of 8 effects. Each position is colour-coded. |
| PARAM 1 | Knob for the first parameter of the selected effect (also CV-controllable). |
| PARAM 2 | Knob for the second parameter of the selected effect. |
| DRY/WET | Blends the dry input signal with the wet (processed) output. Fully CCW = dry only; fully CW = wet only. |

## Inputs

| Jack | Description |
|---|---|
| IN | Mono audio input. Maximum input amplitude: 10 VPTP without clipping. |
| CV | CV input for PARAM 1. ±5 V sweeps the full parameter range. |

## Outputs

| Jack | Description |
|---|---|
| OUT L | Left stereo output. |
| OUT R | Right stereo output. |

## Notes

- The module accepts a mono input and produces stereo output — effects that have stereo character (e.g., stereo delay, Leslie) use both outputs. For mono effects (mono delay, overdrive), both outputs carry the same signal.
- The CV input is summed with the PARAM 1 knob position. Setting the knob to center gives the CV the most bidirectional range.
