| --- | --- |
| Format | Eurorack |
| Width | 16 HP |

## Power

| Rail | Draw |
| --- | --- |
| +12 V | 90 mA (some sources cite 130 mA) |
| -12 V | 70 mA |

## Per VCO — pitch CV inputs

| | |
| --- | --- |
| Jacks | 6 total (3 per VCO), summed |
| Type | 3.5 mm TS, DC coupled |
| Impedance | 50 kΩ unbalanced |
| CV range | ±10 V, 1 V/oct |
| Tracking | ~4% per volt |

## Per VCO — PWM input

## Per VCO — audio outputs

| | |
| --- | --- |
| Jacks | 4 total (2 per VCO) plus SYNC OUT |

| Impedance | 1 kΩ unbalanced |
| Max output level | 10 V p-p |
| Triangle | Bipolar, ±5 V |
| Sawtooth | Unipolar, 0 V to +10 V |

## Frequency range

| | |
| --- | --- |
| With no external CV | 10 Hz to 500 Hz |
| With external CV | 1 Hz to 30 kHz |
| Octave switch | 32′, 16′, 8′, 4′, 2′ |

## Tuning notes

The 112 lacks active temperature compensation on the exponential converter. Expect drift with case temperature. Allow a 10–15 minute warm-up before fine-tuning for studio work.
