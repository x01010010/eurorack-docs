# Modes

## Manual Touch Controller Mode

Touch individual pads to play — each pad instantly outputs its two stored CV values. This is a direct, hands-on performance controller. Assign pitches to the pads in a logical layout (e.g. a scale, a chromatic row, a pentatonic grid) and play melodies or chords by touching pads.

In this mode CELLZ works like a touch keyboard or performance pad with user-programmed pitches. The GATE OUT fires when any pad is touched, making it straightforward to trigger an envelope or VCA from pad activity.

## Arpeggiator Mode

Hold two or more pads simultaneously — CELLZ begins automatically stepping (arpeggiating) between the held pads at the rate set by the SPEED knob. The GATE OUT fires on each step. This turns CELLZ into a simple live arpeggiator: hold any combination of pads to create a looping sequence of those CV values, at the speed you set.

## LFO-Cycled Sequencer Mode

Connect an external LFO signal (or Chipz LFO Out1) to LFO TRIG 1 and optionally Chipz LFO Out2 to LFO TRIG 2. CELLZ advances through its active/programmed pads in response to each trigger, creating an automatic cycling sequence. The rate of the sequence is determined by the external LFO frequency.

With both LFO trigger inputs patched, two independent cycle speeds are running simultaneously — LFO Trig 2 (4× Chipz LFO Out2 = 4× faster) creates a sub-sequence within the main sequence pacing.

## Quantized Pitch Mode

With the QUANTIZE switch on, CV Out 1 values snap to semitone intervals (~2–4 V range). Use this mode when CELLZ is sequencing or controlling a V/Oct pitch input — the quantized mode ensures pads play in-tune notes rather than freely floating voltages.

## Free-Scale CV Mode

With the QUANTIZE switch off, both outputs span the full 0–10 V range with no snapping. Use this for non-pitch CV targets: filter cutoff, VCA level, effect parameters, panning. Programme each pad with precise CV values for parameter automation during performance.
