# Overview

**GHOST** is a 16 HP stereo multi-effects processor from Endorphin.es, developed in collaboration with musician Andrew Huang. It combines delay, reverb, a zero-delay-feedback multimode filter, 8× oversampled distortion, a compressor, and sidechain ducking into a single fully CV-able module — with a one-button routing system that reorders the three main effect blocks on the fly.

The defining feature is the ROUTING button: pressing it cycles through six orderings of DELAY/REVERB, VCF, and DISTORTION. Running distortion before the filter gives aggressive filter sweeps; running the filter before reverb shapes the tail's character. Each routing produces a genuinely different instrument.

Signal enters through pre-VCA gain staging, passes through the ordered effect chain, hits the compressor and sidechain ducking stage, then exits through a post-VCA output stage with a built-in drive reserve. The 96 kHz / 32-bit internal processing (with 16-bit I/O) keeps the signal clean throughout.

## Sonic Territory

- **Ambient washes** — hall reverb with shimmer, pre-delay, and audio freeze.
- **Karplus-Strong synthesis** — the 1V/oct delay with short time and high feedback synthesises pitched resonant strings.
- **Aggressive multi-FX** — distortion into filter into reverb for blown-out textural processing.
- **Rhythmic delay** — tap tempo or external clock with selectable stereo tap patterns.
- **Dynamic drums** — sidechain ducking triggered from an external trigger for pumping effects.
- **Comb filter drones** — the resonator mode turns the filter into a pitched comb for resonant, metallic textures.

## At a Glance

- **16 HP**, 26 mm deep
- **Stereo in/out** (IN 1 normalled to IN 2)
- **3 reorderable effect blocks**: Delay/Reverb, VCF, Distortion (6 routing combinations)
- **Filter modes**: LP/HP, Band-pass, Comb
- **Delay**: up to 2.5 s, 1V/oct, tap tempo, external clock, 3 stereo tap patterns
- **Reverb**: hall with shimmer, reverse, freeze
- **Compressor** + sidechain ducking with envelope output
- **19 patch points**, 13 knobs, 4 buttons
- **96 kHz / 32-bit** internal processing
- **Power**: 135 mA (+12 V) / 35 mA (-12 V)
