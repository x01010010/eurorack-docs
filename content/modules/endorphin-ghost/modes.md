# Modes

## Effect Block Routing

The ROUTING button cycles through three orderings of the main effect blocks. Each press changes the signal chain — the ROUTING LED brightness indicates the current state.

| Routing | Chain | Character |
| --- | --- | --- |
| 1 (LED off) | FX (Delay/Reverb) → Distortion → VCF | Distorted reverb tails, filter shapes post-distortion harmonics |
| 2 (LED half) | Distortion → VCF → FX | Classic dist-into-filter-into-reverb; most "guitar pedal" feel |
| 3 (LED full) | VCF → FX → Distortion | Filter sweeps the clean signal, reverb soaks it, then distortion adds grit |

Routing can be changed live during performance with no audio dropout.

## Filter Modes

Cycle through filter types with the BPF/COMB button.

**LP/HP** (default) — zero-delay-feedback state-variable filter. CCW is low-pass (dark), CW is high-pass (thin). 12 o'clock is neutral. Resonance adds emphasis at the cutoff.

**Band-pass** — narrows the frequency band around the cutoff. RESONANCE controls the bandwidth (higher = narrower).

**Comb** — a resonator/comb filter. Resonance is bipolar: CCW produces negative comb (notches), CW produces positive comb (peaks). FREQUENCY sets the comb's fundamental. Useful for metallic, pitched drone textures.

## Reverb Modes

**Normal** — standard hall reverb with shimmer at longer tail settings. TAIL controls decay, REVERB controls wet/dry mix.

**Hold/Freeze** — press HOLD or gate FREEZE jack. The reverb buffer freezes indefinitely. Audio continues to pass through the other effects. Release to unfreeze.

**Reverse** — hold ROUTING and press HOLD/REVERSE. The reverb plays back in reverse for whooshing, swelling tails. Works best with percussive or rhythmic input material.

## Delay Stereo Tap Patterns

Hold ROUTING and press TAP to cycle through:

- **LRLR** — alternating left/right taps (standard ping-pong).
- **LRRL** — left-right-right-left pattern for a wider, more complex stereo spread.
- **STEREO** — both channels receive identical taps (mono delay, stereo width from reverb).

## Compressor

The compressor is a one-knob single-band peak stereo compressor. It operates on the full mixed signal at the end of the FX chain, before the sidechain ducking and post-VCA stage.

- **CCW**: no compression.
- **12 o'clock**: preset tuned for ambient/pad material.
- **Full CW**: preset tuned for drums and transient-heavy material.
