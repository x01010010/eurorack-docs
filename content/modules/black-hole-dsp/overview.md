# Overview

Black Hole DSP is a stereo multi-effects processor built around the Spin Semiconductor FV-1 DSP chip. Eight factory programs cover hall and room reverbs, delays, chorus, flanger, and pitch shifting. A second bank of eight slots can hold user-written FV-1 programs uploaded via the PROG jack, extending the module well beyond its factory capabilities.

Three parameter knobs (P1, P2, P3) each have a dedicated CV input, giving full voltage control over effect character. Stereo audio input and output make it equally useful as an insert processor or a send/return effect.
