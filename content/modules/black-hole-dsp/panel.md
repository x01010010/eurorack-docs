# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| PROGRAM | Knob | Selects the active effect program (16 slots: 8 factory + 8 user) |
| P1 | Knob | Effect parameter 1 — function depends on selected program |
| P2 | Knob | Effect parameter 2 — function depends on selected program |
| P3 | Knob | Effect parameter 3 — function depends on selected program |
| DRY/WET | Knob | Blend between dry input signal and processed effect output |

## Inputs

| Jack | Function |
| --- | --- |
| IN L | Stereo left audio input |
| IN R | Stereo right audio input (normalled to IN L for mono operation) |
| P1 CV | CV control of parameter 1 |
| P2 CV | CV control of parameter 2 |
| P3 CV | CV control of parameter 3 |
| PROG | Program upload jack — connects to a computer or programmer to load custom FV-1 programs into user slots |

## Outputs

| Jack | Function |
| --- | --- |
| OUT L | Stereo left output |
| OUT R | Stereo right output |
