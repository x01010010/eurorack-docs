# Patch Recipes

## Melodic Resonator Voice

Patch a sequencer V/OCT to V/OCT and its gate to STRUM. Leave IN unpatched. Select the modal or string type. Halos resonates internally on each strum trigger, producing pitched, decaying tones that follow the melody. Adjust DAMPING for note length.

## Exciter Filtering

Patch a noise source or drum machine output into IN. Leave STRUM unpatched. Halos acts as a resonant filter, colouring the noise with its mode series — turns a white noise burst into a tuned metallic ping.

## Reverb-Like Pad

Select the string with reverb type. Set DAMPING high (long decay). Feed a sparse arpeggio into STRUM with V/OCT tracking the melody. The sustained resonator tails layer and blend like a reverb wash. Use ODD/EVEN panned wide for a lush stereo reverb substitute.

## Odd/Even Separation for Processing

Patch ODD to one VCF (tuned bright) and EVEN to another (tuned dark). Mix the processed signals. The harmonic partials respond differently to filtering, creating a complex, evolving timbre that couldn't be achieved from a single output.

## CV Modulation of Structure

Patch a slow LFO to STRUCTURE CV. The resonator transitions between harmonic and inharmonic character over time — useful for evolving pads that gradually drift from musical to metallic. Combine with a slow DAMPING CV sweep for maximum movement.

## Sympathetic String Chord Bed

Select sympathetic strings mode with 4-voice polyphony. Play 4 different chord notes via STRUM triggers with V/OCT changes between each. The four voices sustain and interact, creating a dense sympathetic resonance pad. Add an audio signal to IN for the strings to respond to.
