# Patch Recipes

## Standard Reverb Send/Return

Patch the output of a mixer or VCA into IN L. Take OUT L and OUT R to the left and right channels of your output module. Set DRY/WET fully wet. Control the reverb mix at the mixer send level. This keeps the dry signal completely separate from the effect return.

## Stereo Width from Mono

Patch a mono VCO or synth voice into IN L only (IN R normalls from IN L). Select a chorus or hall reverb program. OUT L and OUT R produce a wide stereo spread from the mono source. Use for pads and leads that need stereo presence without a separate width module.

## CV-Controlled Reverb Decay

Select a hall or plate reverb program. Patch a slow envelope or LFO to P1 CV (typically decay/size). As the envelope opens, the reverb tail extends; as it closes, reverb shortens. Useful for dynamic reverb that responds to note intensity.

## Envelope-Triggered Pitch Shift

Select the pitch shift program. Patch an envelope to P1 CV (shift amount). Set the base pitch shift at the P1 knob (e.g. unison). The envelope pushes the shift upward on each trigger — a momentary octave-up effect that returns to unison as the envelope decays.

## Modulated Delay

Select the echo/delay program. Patch a slow LFO to P1 CV (delay time). The LFO continuously varies the delay time, producing a tape-wobble or chorus-delay hybrid. Set P2 (feedback) moderately high for multiple modulated repeats.

## User Program Exploration

Upload custom FV-1 programs to the user slots for effects unavailable in the factory set — bitcrush, ring modulation, granular shimmer, or custom reverb algorithms. The FV-1 has a wide community of freely available programs. Use the PROG jack with a compatible programmer; factory slots are read-only and unaffected.
