# Overview

**STEPS** is a 14 HP six-stage function generator from Behringer, inspired by the Mutable Instruments Stages architecture. Its defining characteristic is that it has no fixed operating mode — the module reconfigures itself entirely based on how you patch it.

Each of the six segments has a knob, an LED, a gate/trigger input, and contributes to an output. When a gate input is left unpatched, that segment joins the group to its right, forming a multi-stage envelope. When a gate input is patched, it becomes the start of a new independent group. The result is that a single **STEPS** module can simultaneously be a six-stage complex envelope, six independent one-shot segments, a step sequencer with tunable levels, a bank of LFOs, or any combination of these.

The green/orange/red LED color per segment indicates its current type: step (green), ramp (orange), or hold (red), selected by pressing the segment's button while turning its knob. This selection determines whether the segment rises linearly, holds a constant voltage, or jumps immediately — the building block of any shape.

Up to six **STEPS** modules can be chained via a dedicated cable to create sequences and envelopes of up to 36 stages.

## Sonic Territory

- **Complex multi-stage envelopes** — AHDSR, DAHDSR, and beyond, with per-stage curve type.
- **Clocked step sequences** — each segment holds a voltage, advanced by an external clock.
- **Independent LFOs** — ungated segments cycle freely at rates set by their knobs.
- **Cascading function generators** — multiple behaviors from one module in a single patch.
- **Extended sequences** — chain multiple **STEPS** units for long, slowly evolving sequences.
