# Patch Recipes

## Classic Melodic Sequence

Patch CV1 to a VCO V/OCT input. Patch GATE to a VCA envelope trigger. Set step count and tune the 8 CV knobs to the desired melody. Use TEMPO to control speed. A complete self-contained melodic voice with one patch cable to VCO and one to envelope.

## Two-Voice Counterpoint

Run the 182 in 8-step parallel mode (SERIES off). Patch CV1 to VCO 1 and CV2 to VCO 2. Both voices step simultaneously but follow different pitches. Set CV1 for a bass line and CV2 for a mid-range counter melody.

## 16-Step Melody

Enable SERIES. CV1 now outputs a 16-step sequence combining both rows. Patch to a single VCO for a longer phrase before looping.

## Rhythmic Gating with Gate Time

Patch GATE to a VCA. Reduce GATE TIME to make short staccato notes. Increase GATE TIME for legato, sustained notes. Automate GATE TIME with a slow LFO for expressive phrasing variation.

## Pendulum Sequence

Set MODE to Pendulum. The sequence plays forward and then backward, doubling the perceived length before repeating. Tune the first and last step to the same pitch for smooth transitions.

## External Clock Sync

Patch a master clock (from a drum machine or clock module) to EXT CLOCK. The 182 locks to the external tempo. Use RESET to sync to downbeat. Useful for synchronising with other sequencers in a larger system.
