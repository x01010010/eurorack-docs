# Overview

The 182 is a dual-channel 8/16-step CV sequencer based on the Roland System 100M 182 Sequencer. It provides two independent CV rows of up to 8 steps each, which can be linked in series for a 16-step sequence. Each row has individual step knobs for precise CV programming.

Clocking, gate time, step count, and playback mode are all front-panel adjustable. Dedicated CV, gate, and trigger outputs allow flexible integration with VCOs, VCFs, VCAs, and other modules. An internal clock with TEMPO control means the 182 can drive a patch without any external clock source.
