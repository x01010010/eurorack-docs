# Specifications

## Electrical

| Parameter | Value |
| --- | --- |
| Frequency range | 16 Hz – 32 kHz |
| Waveforms | 7 (continuous morphing via waveshaper) |
| Output voltage | 0–5 V (PULSE and OUT) |
| 1V/OCT tracking | 1 V per octave |
| Power (+12 V) | 58 mA |
| Power (-12 V) | 15 mA |
| Power (+5 V) | Not required |

## Mechanical

| Parameter | Value |
| --- | --- |
| Width | 10 HP |
| Depth | 25 mm |
| Connector | 16-pin Eurorack power header |

## Calibration

The v1.1 revision uses digital trimpots with a simplified auto-tuning procedure — no screwdriver needed. Calibration can be adjusted per CV source for tighter tracking with your specific sequencer or keyboard.
