# Tips & Tricks

- **Pulse width as a timbral tool** — even when using the main OUT, try patching the PULSE output into a mixer alongside it. Blending a narrow pulse with the shaped waveform adds presence and bite.
- **Use the quantizer as a performance toggle** — map a gate or button to nothing and just tap the QUANTIZE button live. Switching between quantized steps and continuous glide mid-performance creates expressive pitch effects.
- **Sync sweep sweet spot** — when using hard sync, the most dramatic timbral change happens when Hysteria is tuned above the sync source. Try a slow envelope on the CV input rising from unison to a fifth above.
- **Ultra-low LFO** — turn OCT all the way down in unquantized mode. At 16 Hz and below (with negative CV), Hysteria produces slow waveforms usable as a modulation source. The PULSE output becomes a clock at that rate.
- **Pair with an external quantizer for scales** — the built-in quantizer is chromatic only (all 12 semitones). For scale-constrained melodies, bypass Hysteria's quantizer and use an external module set to your desired scale.
- **Shallow depth advantage** — at 25 mm deep, Hysteria fits in skiff cases and portable setups where deeper modules won't. Good candidate for a travel rack's primary voice.
- **V1.1 improvements** — if you have an older v1.0 unit, note that CV inputs were inverted on that revision. V1.1 corrected this to the standard non-inverted convention.
