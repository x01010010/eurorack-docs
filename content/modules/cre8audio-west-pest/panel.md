# Panel

The panel is organised left-to-right in functional blocks.

## Oscillator

- **PITCH** — coarse frequency knob (wide range, self-oscillation capable)
- **MOD** — modulation depth for the multi-mod tool input
- **1V/OCT** — CV pitch input jack
- **AUDIO IN** — external audio input (normalled to oscillator out internally)
- **WAVE** — waveshape select (sine, triangle, sawtooth, pulse)
- **OSC OUT** — oscillator output jack (pre-folder)

## Wavefolder

- **FOLD** — wavefold amount knob (up to six folds)
- **FOLD CV** — CV input for fold amount
- **FOLD OUT** — post-fold audio output jack

## Dynamics Controller

- **FREQUENCY** — sets the filter cutoff / decay speed
- **RESONANCE** — adds ringing resonance to the decay
- **STRIKE** — manual trigger button
- **TRIGGER IN** — gate/trigger input jack
- **AUDIO IN** — input jack (normalled to fold out)
- **OUT** — dynamics controller output jack

## LFO

- **RATE** — LFO speed knob
- **WAVE** — shape select (triangle / square)
- **LFO OUT** — LFO output jack

## Multi-Mod Tool

- **MODE** — selector switch: LFO sync, decay envelope, MIDI mod, random
- **RATE/DEPTH** — context-sensitive control
- **OUT** — multi-mod output jack

## MIDI / CV

- **MIDI IN** — 5-pin DIN socket
- **CV OUT** — 1 V/oct pitch output jack
- **GATE OUT** — gate output jack

## Sequencer / Keyboard

- 8 step-program buttons with LED indicators
- 13 keyboard buttons (one octave + one)
- **TEMPO** — internal clock rate knob
- **CLOCK IN** — external clock input jack
- **RUN/STOP** — sequencer transport button
