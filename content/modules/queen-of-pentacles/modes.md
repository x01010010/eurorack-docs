# Modes

## Sample Channel Knob Mode (S1–S4)

Each sample channel's LEVEL knob has two modes, toggled by pressing the MODE button:

- **Decay mode** (blue LED): the knob controls the sample's volume envelope decay time — how long the sample rings out after the trigger.
- **Pitch mode** (red LED): the knob controls sample playback pitch. CCW slows and lowers pitch, CW speeds and raises pitch.

Long-pressing the MODE button (2 seconds) cycles through the 8 available sample banks.

## Effect Processor Banks

The effect processor has two banks of 8 effects each, selected via the TYPE button. The HAL9000 LED indicates the current selection:

**Airways Bank (reverbs/delays):**

| # | Effect |
| --- | --- |
| 1 | Hall Reverb |
| 2 | Shimmer Reverb |
| 3 | Stereo Room Reverb |
| 4 | Plate Reverb |
| 5 | Spring Reverb |
| 6 | Ping-Pong Delay |
| 7 | Tape Echo |
| 8 | Chorus |

**Darkwaves Bank (modulation/dynamics/loops):**

| # | Effect |
| --- | --- |
| 1 | Gated Reverb |
| 2 | Spring Reverb |
| 3 | Reversed Reverb |
| 4 | Flanger |
| 5 | Ring Modulator |
| 6 | Overdrive |
| 7 | Peak Compressor |
| 8 | Freezer / Looper |

CABIN FEVER's secondary function varies by effect — for delay-type effects it controls feedback; for reverbs it controls decay time; for modulation effects it may control rate or depth.

## Channel Effect Routing

Each channel has a three-position switch:

- **Left** (effect off): channel passes to the main mix dry.
- **Centre** (mute): channel is silenced entirely from the main mix output.
- **Right** (effect on): channel is routed through the effect processor before the main mix.

Individual output jacks bypass the mix bus entirely — a channel with a cable in its OUT jack is removed from the stereo master mix regardless of the switch position.

## Rolls

When an external clock is patched to the CLOCK IN jack, holding any manual TRIG button triggers a roll on that channel — rapid repeated triggers following the incoming clock rate. Releasing the button stops the roll.

## Force Effect on All Channels

Holding TAP for more than one second enables the effect processor on all channels simultaneously, overriding their individual switches. Useful for a quick global wet effect during performance. Hold again to revert.
