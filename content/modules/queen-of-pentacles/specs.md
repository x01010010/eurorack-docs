# Specifications

## Electrical

| Parameter | Value |
| --- | --- |
| Analog voices | 3 (BD, SD, CP) |
| Sample voices | 4 (S1–S4) |
| Sample storage | microSD card |
| Sample banks | 8 (factory + user) |
| Sample playback latency | Zero-latency (direct from SD) |
| Effect processor | 16 effects (2 banks × 8) |
| Noise generators | Band-limited LSRF binary, per analog voice |
| Power (+12 V) | 350 mA |
| Power (−12 V) | 90 mA |
| Power (+5 V) | Not required |

## Mechanical

| Parameter | Value |
| --- | --- |
| Width | 30 HP |
| Depth | 26 mm (with power cable) |
| Connector | 16-pin Eurorack power header |

## I/O Summary

| Type | Count |
| --- | --- |
| Trigger inputs | 7 (one per voice) |
| Accent/velocity inputs | 7 (one per voice) |
| Individual audio outputs | 7 (BD, SD, CP, S1/S2 pair, S3/S4 pair) |
| Stereo master outputs | 2 (L/R) |
| Effect CV input | 1 (Cabin Pressure) |
| Aux input | 1 |
| Clock input | 1 |
| Filter CV input | 1 |
