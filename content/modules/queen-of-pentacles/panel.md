# Panel & I/O

## Per-Channel Controls & Jacks

### Bass Drum (BD)

| Control/Jack | Function |
| --- | --- |
| DRIVE knob | Input drive/distortion amount for the analog BD circuit. |
| TUNE knob | Pitch of the bass drum. |
| DECAY knob | Body decay time. |
| TAIL DECAY knob | Tail/sub decay time (secondary ring). |
| SPOILER knob | Noise sample rate — adjusts noise spectrum animation. |
| TRIG button | Manual trigger button. |
| TRIG IN jack | Trigger/gate input (0–+5 V). |
| ACCENT IN jack | Velocity/accent CV input. |
| OUT jack | Individual audio output. Patching removes BD from the main mix. |
| Effect switch | Left = effect off, centre = mute, right = effect on (affects main mix only). |

### Snare Drum (SD)

| Control/Jack | Function |
| --- | --- |
| TUNE knob | Snare pitch. |
| TAIL DECAY knob | Snare decay time. |

| TRIG button | Manual trigger. |

### Clap (CP)

| Control/Jack | Function |

| TRIG button | Manual trigger. |

### Sample Channels S1–S4

| Control/Jack | Function |
| --- | --- |
| LEVEL knob | Channel volume. In decay mode (blue LED): sets decay time. In pitch mode (red LED): sets sample pitch. |
| MODE button | Short press: toggle between decay (blue) and pitch (red) control for LEVEL knob. Long press (2 s): cycle through sample banks 1–8. |
| TRIG button | Manual trigger. |
| TRIG IN jack | Trigger/gate input. |
| ACCENT IN jack | Velocity/accent CV input. |
| OUT jack | S1/S2 share a stereo output pair; S3/S4 share a stereo output pair. Patching removes that pair from the main mix. |
| Effect switch | Effect enable/mute/off. |

## Effect Processor (Cabin Pressure)

| Control/Jack | Function |
| --- | --- |
| CABIN PRESSURE knob | Dry/wet mix for the effect (CCW = dry, CW = wet). Acts as CV attenuator when a cable is patched. |
| CABIN FEVER knob | Secondary parameter — varies by effect type (decay, feedback, hi-pass, etc.). Activated by holding TAP. |
| TAP button | Short press: cycles effect type within current bank. Long hold (>1 s): enables effect on all channels regardless of individual switches. Hold 5+ s: alternate routing mode. |
| TYPE button | Selects effect number within bank (HAL9000 LED blinks to indicate: green 1–4×, red 1–4×). |
| CABIN PRESSURE CV jack | −5–+5 V CV for dry/wet mix. |
| AUX IN jack | External audio input for the effect processor. Also used for firmware updates via audio. |

## Master Section

| Control/Jack | Function |
| --- | --- |
| FILTER knob | Bipolar LP/HP isolator filter (CCW = low-pass, CW = high-pass, 12 o'clock = flat). |
| FILTER CV jack | CV input for filter cutoff. |
| VOLUME knob | Master output level. Adds saturation boost past approximately 2 o'clock. |
| OUT L / OUT R jacks | Stereo master output. |
| CLOCK IN jack | External clock input for roll functions on all channels. |
