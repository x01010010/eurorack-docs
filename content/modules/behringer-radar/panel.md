# Panel & I/O

## Controls

| Control | Type | Function |
| --- | --- | --- |
| GAIN | Knob | Input gain for the contact mic signal |
| ATTACK | Switch | Sets the attack time of the envelope follower (fast/slow) |
| DECAY | Switch | Sets the decay time of the envelope follower (fast/slow) |
| SENSITIVITY | Switch | Sets the threshold sensitivity for gate detection (low/high) |

## Inputs

| Jack | Function |
| --- | --- |
| MIC IN | Contact microphone input (3.5 mm or 6.35 mm depending on version) |

## Outputs

| Jack | Function |
| --- | --- |
| AUDIO OUT | Amplified and conditioned audio signal from the contact mic |
| ENV OUT | Envelope follower CV output (0–10 V) — tracks the amplitude of the input signal |
| GATE OUT | Gate output (0–5 V) — goes high when input exceeds the SENSITIVITY threshold |
