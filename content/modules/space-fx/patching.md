# Patch Recipes

## Standard Reverb Send

Patch the output of a VCA or mixer into FX SEND. Take OUT L and OUT R to a stereo mixer or two channels of a case output. Set PROGRAM to a hall or plate reverb. Set LEVEL to mix. This is the classic send/return reverb setup.

## Ping-Pong Delay on a Sequence

Select the ping-pong delay algorithm. Tap the TAP button in time with your sequencer tempo to set delay time. Patch a mono melodic line in — echoes bounce between left and right outputs, adding rhythmic depth and stereo width to a mono sequence.

## Spring Reverb for Drum Transients

Select the spring reverb algorithm. Send a snare or hi-hat through FX SEND. The spring algorithm adds the characteristic boing and resonance of a physical spring, thickening the transient. Use a short reverb time for an 80s drum machine character.

## Shimmer Reverb Pad

Select the shimmer reverb algorithm. Patch a sustained note or pad into FX SEND. The shimmer algorithm pitch-shifts the reverb tail upward, creating an angelic, ethereal pad that rises and sustains. Works best with slow attack, sustained sounds.

## Chorus for Stereo Width

Select a chorus or ensemble algorithm. Patch a mono oscillator into FX SEND. OUT L and OUT R produce a wide stereo field from the mono source. Use subtly for string pad width, or more deeply for classic chorus effects.

## Full Wet Effects Voice

Set LEVEL to maximum wet. Patch a drum machine or sequencer into FX SEND. Everything you hear is only the effect — use with a heavily feedback delay or a massive reverb to create drone and pad textures from percussive sources.
