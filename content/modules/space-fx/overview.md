# Overview

Space FX is a stereo multi-effects processor built on Klark Teknik DSP technology, offering 32 algorithms including reverbs, delays, choruses, flangers, phasers, pitch shifters, and combination effects. An 8-character display, PROGRAM knob, and two parameter knobs allow direct access to any algorithm and its key parameters.

The module operates as a parallel effects send — the main signal path remains dry while the wet signal appears at the output, with a WET/DRY mix level switch allowing full wet or blended operation. Space FX is designed for Eurorack integration with standard ±12V power and mono input with stereo output.
