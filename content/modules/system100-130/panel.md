# Panel & I/O

The module has two identical VCA channels arranged side by side. Controls and jacks are described per channel.

## Controls (per channel)

| Control | Type | Function |
| --- | --- | --- |
| SIG IN 1–3 knobs | Rotary (×3) | Individual gain for each of the three signal inputs. Sets how much each source contributes to the VCA input. |
| INITIAL GAIN | Rotary | Sets the base gain level of the VCA with no CV applied. Fully CCW = silence; fully CW = maximum gain. |
| MOD IN 1–3 knobs | Rotary (×3) | Individual attenuators for each of the three modulation (CV) inputs. |
| LIN / EXP | Slide switch | Selects linear or exponential VCA response curve. |

## LEDs (per channel)

| LED | Function |
| --- | --- |
| Signal LED | Lights when a signal is present at the output. |
| Clip LED | Lights when the output is overloading — reduce SIG IN gain or INITIAL GAIN. |

## Inputs (per channel)

| Jack | Function |
| --- | --- |
| SIG IN 1 | Audio/signal input 1 (3.5 mm TS). |
| SIG IN 2 | Audio/signal input 2 (3.5 mm TS). |
| SIG IN 3 | Audio/signal input 3 (3.5 mm TS). |
| MOD IN 1 | Modulation (CV) input 1 — controls VCA gain (3.5 mm TS). |
| MOD IN 2 | Modulation (CV) input 2 — controls VCA gain (3.5 mm TS). |
| MOD IN 3 | Modulation (CV) input 3 — controls VCA gain (3.5 mm TS). |

## Outputs (per channel)

| Jack | Function |
| --- | --- |
| HIGH OUT | Line-level output (1 kΩ impedance, +18 dBu max). For connecting to line-level mixers, powered speakers, or other line-level gear. |
| LOW OUT | Instrument-level output (3 kΩ impedance, −2 dBu max). For guitar amplifiers or instrument-level inputs. |

## Summed Outputs

The HIGH OUT and LOW OUT on each channel also act as summed mix outputs — both VCA channels are summed and available at both output jacks. This allows the single pair of outputs to carry a blend of both channels.

## Panel Notes

- CV input range is 0 V to +10 V. Negative voltages will not reduce gain below the INITIAL GAIN setting.
- In exponential mode, CV scaling is approximately 1 V per 10 dB.
- Input impedance for all jacks is >50 kΩ.
