# Overview

The **297** is Behringer's reproduction of the Roland System 100M model 297, combining a dual portamento/slew controller with two independent CV mixing stages in a single 16 HP panel.

Each portamento channel provides independent RISE and FALL controls so glide time can be set asymmetrically — slow up, fast down, or vice versa. The circuit responds to incoming CV or gate transitions, making it equally useful for pitch glide, CV smoothing, and simple function generation.

The two CV mixer sections each accept two inputs with individual attenuators and sum them to a single output. Combined with the portamento channels, the 297 can smooth a mixed CV signal in a single patch.

Like all System 100 modules, the 297 follows the Roland 100M panel aesthetic: compact, legible, and straightforwardly laid out. It pairs naturally with the **112** VCO, **121** filter, and **140** envelope when building a complete voice.
