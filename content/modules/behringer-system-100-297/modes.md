# Modes

The 297 has no mode switches — all four sections operate simultaneously and independently. However, the portamento channels are flexible enough to serve several distinct roles depending on how they are patched.

## Standard Portamento

Feed a sequencer's 1 V/oct output into Portamento IN. RISE and FALL set the glide speed for ascending and descending intervals independently. Patch OUT to a VCO.

## Equal-Time Glide

Set RISE and FALL to the same value for classic, symmetrical glide with equal up/down transition times.

## Fast Attack, Slow Decay (AR-style)

Set RISE to minimum and FALL to a longer time. When triggered by a stepped CV (e.g., from a gate-keyed sequencer), the portamento channel produces a fast-attack, slow-decay shape — a simple AR envelope substitute for CV control.

## CV Smoothing / Lag Processor

Patch any control source (LFO, random, expression) into IN and use OUT as a smoothed, lag-processed version. Useful for softening abrupt modulation sources before applying them to sensitive destinations.

## CV Mixing

The two CV mixer sections operate independently. Use them to sum pitch offsets, combine LFOs, or scale down modulation amounts before mixing.
