# Overview

**BLACK STEREO MIXER V2** is a 10 HP performance stereo mixer from Erica Synths, designed for racks that have accumulated multiple stereo modules — effects processors, granular engines, stereo oscillators — and need a compact way to sum them before the output stage.

Four inputs are arranged with different characters. Three accept stereo pairs with a single ganged knob controlling both L and R levels simultaneously, so balancing a stereo channel never requires touching two controls. The fourth input is mono with a dedicated CV input for automated stereo panning — useful for dynamic placement of a mono signal in the stereo field under sequencer or LFO control. Two of the inputs additionally support an L+R channel merge mode, allowing mono signals arriving on a stereo input to be summed to both channels. A switchable +6 dB boost on IN1 and IN2 compensates for modules that output at lower-than-expected levels. Signal level LEDs on the L and R buses give a visible indication of the mix level without needing an external meter.

The module is a passive-sum design with unity gain and is compatible with the optional Black Pre-fade Listen (PFL) expander.
