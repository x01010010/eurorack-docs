# Patch Recipes

## Four-Voice Drum Machine

Patch TRIG 1–4 to four different drum modules (kick, snare, hi-hat, percussion). Design a 16-step pattern in the web editor with a standard kick/snare grid and off-beat hats. Upload and use the module as a self-contained drum sequencer.

## Polyrhythmic Evolving Pattern

Design a pattern where each track has a different step length (e.g., 3, 5, 7, 11). The four tracks phase against each other, creating a pattern that takes hundreds of steps to fully repeat. Patch to percussive voices or envelope triggers for generative rhythms.

## Clock Source for a Small System

Use CLK OUT as the master tempo for your system. Patch it to sequencer clock inputs, clock dividers, or delay sync. Adjust tempo with the encoder during performance.

## Shuffled Groove

Upload a straight 16-step pattern, then dial in shuffle with the encoder. Even a small amount of shuffle transforms a mechanical grid into a more human-feeling groove. Works well driving hi-hat and percussion modules.

## Trigger-to-Envelope Chains

Patch each TRIG output through an envelope generator (like the Black EG2) before reaching your sound source. Each track gets its own envelope shape — snappy for drums, slow for pads — all from the same trigger pattern.

## External Clock Sync

Patch your main system clock into CLK. The pattern now follows your existing tempo. Use this to add polyrhythmic trigger layers on top of a sequence you're already running from another module.
