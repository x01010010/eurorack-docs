# Overview

**PICO TRIGGER** is a 3 HP polyrhythmic trigger generator and master clock from Erica Synths. Despite its tiny footprint, it provides 4 independent trigger outputs running patterns up to 16 steps long, with per-track step lengths enabling polyrhythmic combinations. It stores 8 patterns and includes adjustable shuffle.

Patterns are designed in a web-based interface on your computer or phone, then uploaded to the module via the CLK jack using an audio cable. This unusual approach means the module's panel can stay minimal — just an encoder and a button — while offering deep pattern editing through the browser tool.

## Sonic Territory

- **Polyrhythmic drum triggers** — set each of the 4 tracks to different step lengths (e.g., 3, 5, 7, 16) for evolving, non-repeating rhythmic interplay.
- **Simple clock divisions** — use built-in divider patterns to clock other modules at /2, /4, /8, etc.
- **Shuffled grooves** — adjustable shuffle adds swing to otherwise straight patterns.
- **Master clock source** — drives an entire small system from 20–420 BPM via the clock output.
- **Pattern switching** — load different patterns on the fly for live performance variation.

## At a Glance

- **3 HP** Eurorack module, 35 mm deep
- **4 trigger outputs** with independent patterns up to 16 steps
- **Polyrhythmic** — per-track variable step length
- **8 pattern memory** slots, switchable live
- **Adjustable shuffle**
- **Internal clock** (20–420 BPM) or external clock input
- **Clock output** for syncing other modules
- **Web-based pattern designer** — upload via CLK jack
- **Power**: 20 mA (+12 V) / 5 mA (-12 V)
