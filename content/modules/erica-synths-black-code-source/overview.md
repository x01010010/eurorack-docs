# Overview

**BLACK CODE SOURCE** is a 10 HP stereo digital noise and dual random CV generator from Erica Synths. Rather than simple sample-and-hold randomness, it generates its outputs through polynomial feedback calculations — the same mathematical technique used in vintage digital drum machines, arcade game audio chips, and data scrambling. The result is noise and pseudo-random CV with characteristic digital texture, including a ZX Spectrum tape-loading mode that recreates the sound of an 8-bit home computer loading software.

The module's core controls are four knobs: RATE sets the clock (and therefore the character of the output — slow clock = stepped CV, fast clock = noise), while X, Y, and S select polynomial configurations and sequence start points from a 16×16 grid. This gives over 200 polynomial combinations and, combined with the three-position MODE switch, more than 4,000 distinct dual CV sequences.

At high RATE values the outputs become stereo digital noise useful for percussion and extreme audio processing. At low RATE values, the same polynomials produce slowly evolving paired CV signals for modulation. The boundary between the two is itself a rich sonic zone.

## Sonic Territory

- **Digital percussion noise** — hi-hat, snare, and noise sweep sounds from the fast-rate region.
- **Dual stepped CV** — slow rate produces two correlated but different random voltage sequences for generative pitch and timbre modulation.
- **ZX Spectrum bleeps** — mode 3 generates tonal, data-encoding noise reminiscent of 8-bit computer loading sounds.
- **Bitcrushing / sound destruction** — patch audio into S TRIG and route outputs back into your signal chain for extreme digital mangling.
- **Repeatable randomness** — sequences can be "rewound" to their start using S and the S TRIG input, making long pseudo-random runs reproducible.

## At a Glance

- **10 HP**, 35 mm deep
- **Dual outputs** (OUT A, OUT B) with independent polynomial sequences
- **RATE** sweeps from slow dual CV to digital noise
- **X × Y**: 16×16 polynomial selection (256 combinations)
- **S**: sequence start point control
- **MODE switch**: 3 positions, each offering 4,000+ sequences
- **CLOCK IN**: external clock replaces internal rate
- **S TRIG**: resets sequence / accepts audio for extreme processing
- All parameters CV controllable with onboard attenuators for X, Y, S
- **Expander-ready** via ribbon cable connector
- **Power**: 143 mA (+12 V) / 64 mA (−12 V)
