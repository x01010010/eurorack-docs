# Tips & Tricks

## Save Presets for Different Performance Contexts

**BITBOX MK2** supports multiple saved presets, each with its own clip assignments, CV mappings, and effects settings. Create one preset for a drum machine setup, another for granular textures, and another for melodic one-shots. Switching presets between sets is faster than re-patching.

## Use a Fast microSD Card

Streaming large samples in real time places demands on the microSD card's read speed. Use a card rated at Class 10 / UHS-I or better. Slow cards can cause dropouts or playback glitches with large files or many simultaneous clips.

## CV Assignment Strategy

With only 8 CV inputs and many assignable parameters, prioritize inputs for the controls you adjust most often in performance — typically clip triggers and pitch. Leave less-used parameters (attack, granular density) on fixed knob positions or controlled from MIDI instead.

## MIDI Type A vs Type B

1010music uses TRS MIDI Type A. Some modules and adapters use Type B (Novation, Make Noise). Using the wrong type will result in no MIDI response. If MIDI isn't working, try a Type B adapter before assuming a fault.

## Granular Mode as a Live Effect

You don't have to pre-load samples for granular processing. Record a live input into a clip slot, immediately enable granular mode, and begin manipulating grain parameters. This creates a real-time granular effect on any incoming audio — similar to using a dedicated granular module but with the additional control of the clip grid.

## Explore Fxbox Before Buying a Reverb Module

Before spending HP on a dedicated reverb or delay module, try the fxbox firmware. It is free, sounds good, and takes advantage of the touchscreen for parameter control. If it meets your needs, you've saved rack space. If not, you'll have a clearer picture of what you actually need from a dedicated module.
