# Overview

**MODEL D** is Behringer's reproduction of the classic Moog Model D (Minimoog) architecture, released in 2017 as one of the first high-profile affordable analog synth clones. It faithfully recreates the three-oscillator, ladder-filter design that defined the sound of countless records, in a 70 HP Eurorack-compatible format.

The three VCOs each offer five waveforms with continuous blending and individual pulse-width control. OSC 3 can be switched from tracking pitch to free-running, enabling it to act as a modulation source — the same trick that made Minimoog patches so lively. A 24 dB/oct transistor ladder filter with resonance and switchable LP/HP modes forms the core of the sound. An overdrive circuit adds saturation before the VCA.

The internal signal path mirrors the Minimoog: OSCs → Mixer (with noise and external in) → VCF → VCA → Output. Modulation routing follows the same logic: the wheel/LFO destination is switchable between OSC pitch and VCF cutoff, with dedicated modulation wheel depth control.

Eurorack compatibility means all key signal and CV points are accessible on 3.5 mm jacks: pitch CV, gate, individual OSC outputs, filter inputs, envelope outputs, and more. A 16-voice MIDI poly chain allows multiple Model D units to be linked for polyphonic performance. Patch points make it a flexible voice module even without MIDI.
