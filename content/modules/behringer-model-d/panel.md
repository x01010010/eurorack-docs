# Panel

The Model D panel follows the classic Minimoog layout with additional Eurorack patch jacks along the top and bottom edges.

## Oscillator Bank

Three identical VCO sections, each with:

- **RANGE** — footage selector (32′ to 1′ in octave steps)
- **WAVE** — waveform selector (triangle, shark-fin, sawtooth, pulse, square)
- **TUNE** — fine-tune ±1 semitone

**OSC 3** additionally has a **FREQ** switch to disengage keyboard tracking (free-running mode for use as LFO/mod source) and a **KB CTRL 3** switch.

## Mixer

- Level knobs for: OSC 1, OSC 2, OSC 3, Noise, External Input
- ON switches per source
- **EXT IN** jack

## VCF

- **CUTOFF** — filter frequency
- **EMPHASIS** — resonance (self-oscillates at maximum)
- **CONTOUR** — filter envelope depth
- **FILTER MOD** switch — enables OSC 3 and/or LFO to modulate cutoff
- **LP / HP** switch

## Envelopes

- **VCF CONTOUR** — ADSR for filter (Attack, Decay, Sustain)
- **LOUDNESS CONTOUR** — ADSR for VCA (Attack, Decay, Sustain)
- Both share a **DECAY/RELEASE** switch for selecting whether decay time also controls release

## VCA / Output

- **MASTER VOLUME** — main output level
- **HEADPHONES** output with dedicated level
- Low and High level outputs on jacks

## LFO / Modulation

- **RATE** — LFO speed
- **WAVE** — triangle or square
- **MOD WHEEL** range control
- **MOD DEST** — routes wheel to OSC pitch or VCF cutoff
- **OSC 3 MOD** — routes OSC 3 as mod source

## Eurorack Patch Jacks

Along the top and bottom: 1V/oct, Gate, Trigger, individual OSC outs, VCF in, VCF out, ENV 1/2 out, LFO out, Noise out, and more.
